# СИ║С╗ђС╣ѕУдЂтГдShell№╝Ъ

тГдСИђСИфСИюУЦ┐№╝їТѕЉС╗гтцДжЃетѕєТЃЁтєхжЃйТў»тЙђт«ъућеТђДТќ╣тљЉуЮђТЃ│сђѓС╗јтиЦСйюУДњт║дТЮЦУ«▓№╝їтГдС╣а Shell Тў»СИ║С║єТЈљжФўТѕЉС╗гУЄфти▒тиЦСйюТЋѕујЄ№╝їТЈљжФўС║ДтЄ║№╝їУ«ЕТѕЉС╗гтюеТЏ┤т░ЉуџёТЌХжЌ┤т«їТѕљТЏ┤тцџуџёС║ІТЃЁсђѓ

тЙѕтцџС║║С╝џУ»┤ Shell у╝ќуеІт▒ъС║јУ┐љу╗┤Тќ╣жЮбуџёуЪЦУ»єС║є№╝їт║ћУ»ЦТў»У┐љу╗┤С║║тЉўТЮЦтЂџ№╝їТѕЉС╗гтЂџтљјуФ»т╝ђтЈЉуџёТ▓Ат┐ЁУдЂтГдсђѓТѕЉУДЅтЙЌУ┐ЎуДЇУ»┤Т│ЋтцДжћЎуЅ╣жћЎ№╝їуЏИТ»ћС║јСИЊжЌетЂџLinuxУ┐љу╗┤уџёС║║тЉўТЮЦУ»┤№╝їТѕЉС╗гт»╣ Shell у╝ќуеІТјїТЈАуеІт║дуџёУдЂТ▒ѓУдЂТ»ћС╗ќС╗гСйј№╝їСйєТў»shellу╝ќуеІС╣ЪТў»ТѕЉС╗гт┐ЁжА╗УдЂТјїТЈАуџё№╝Ђ

уЏ«тЅЇLinuxу│╗у╗ЪСИІТюђТхЂУАїуџёУ┐љу╗┤УЄфтіетїќУ»ГУеђт░▒Тў»ShellтњїPythonС║єсђѓ

СИцУђЁС╣ІжЌ┤№╝їShellтЄаС╣јТў»ITС╝ЂСИџт┐ЁжА╗Сй┐ућеуџёУ┐љу╗┤УЄфтіетїќу╝ќуеІУ»ГУеђ№╝їуЅ╣тѕФТў»тюеУ┐љу╗┤тиЦСйюСИГуџёТюЇтіАуЏЉТјДсђЂСИџтіАт┐ФжђЪжЃеуй▓сђЂТюЇтіАтљ»тіетЂюТГбсђЂТЋ░ТЇ«тцЄС╗йтЈітцёуљєсђЂТЌЦт┐ЌтѕєТъљуГЅуј»УіѓжЄї№╝їshellТў»СИЇтЈ»у╝║уџёсђѓPython ТЏ┤жђѓтљѕтцёуљєтцЇТЮѓуџёСИџтіАжђ╗УЙЉ№╝їС╗ЦтЈіт╝ђтЈЉтцЇТЮѓуџёУ┐љу╗┤Уй»С╗ХтиЦтЁи№╝їт«ъуј░жђџУ┐ЄwebУ«┐жЌ«уГЅсђѓShellТў»СИђСИфтЉйС╗цУДБжЄітЎе№╝їУДБжЄіТЅДУАїућеТѕиТЅђУЙЊтЁЦуџётЉйС╗цтњїуеІт║ЈсђѓСИђУЙЊтЁЦтЉйС╗ц№╝їт░▒уФІтЇ│тЏът║ћуџёС║цС║њуџёт»╣У»ЮТќ╣т╝Јсђѓ

тЈдтцќ№╝їС║єУДБ shell у╝ќуеІС╣ЪТў»тцДжЃетѕєС║њУЂћуйЉтЁгтЈИТІЏУЂўтљјуФ»т╝ђтЈЉС║║тЉўуџёУдЂТ▒ѓсђѓСИІтЏЙТў»ТѕЉТѕфтЈќуџёСИђС║ЏуЪЦтљЇС║њУЂћуйЉтЁгтЈИт»╣С║ј Shell у╝ќуеІуџёУдЂТ▒ѓсђѓ

  1. ТѕЉС╗гУЄфти▒т«џС╣ЅуџётЈўжЄЈ№╝ѕУЄфт«џС╣ЅтЈўжЄЈ№╝Ѕ: С╗ЁтюетйЊтЅЇ Shell т«ъСЙІСИГТюЅТЋѕ№╝їтЁХС╗ќ Shell тљ»тіеуџёуеІт║ЈСИЇУЃйУ«┐жЌ«т▒ђжЃетЈўжЄЈсђѓ
  2. Linuxти▓т«џС╣Ѕуџёуј»тбЃтЈўжЄЈ№╝ѕуј»тбЃтЈўжЄЈ№╝ї СЙІтдѓ№╝џ$PATH, $HOME уГЅ..., У┐Ўу▒╗тЈўжЄЈТѕЉС╗гтЈ»С╗ЦуЏ┤ТјЦСй┐уће№╝Ѕ№╝їСй┐уће env тЉйС╗цтЈ»С╗ЦТЪЦуюІТЅђТюЅуџёуј»тбЃтЈўжЄЈ№╝їУђїsetтЉйС╗цТЌбтЈ»С╗ЦТЪЦуюІуј»тбЃтЈўжЄЈС╣ЪтЈ»С╗ЦТЪЦуюІУЄфт«џС╣ЅтЈўжЄЈсђѓ
  3. ShellтЈўжЄЈ №╝џShellтЈўжЄЈТў»ућ▒ Shell уеІт║ЈУ«Йуй«уџёуЅ╣Т«ітЈўжЄЈсђѓShell тЈўжЄЈСИГТюЅСИђжЃетѕєТў»уј»тбЃтЈўжЄЈ№╝їТюЅСИђжЃетѕєТў»т▒ђжЃетЈўжЄЈ№╝їУ┐ЎС║ЏтЈўжЄЈС┐ЮУ»ЂС║є Shell уџёТГБтИИУ┐љУАї

тИИућеуџёуј»тбЃтЈўжЄЈ:

Сй┐уће Linux ти▓т«џС╣Ѕуџёуј»тбЃтЈўжЄЈ№╝џ

Т»ћтдѓТѕЉС╗гУдЂуюІтйЊтЅЇућеТѕиуЏ«тйЋтЈ»С╗ЦСй┐уће№╝џecho $HOMEтЉйС╗ц№╝ЏтдѓТъюТѕЉС╗гУдЂуюІтйЊтЅЇућеТѕиShellу▒╗тъІ тЈ»С╗ЦСй┐ућеecho $SHELLтЉйС╗цсђѓтЈ»С╗ЦуюІтЄ║№╝їСй┐ућеТќ╣Т│ЋжЮътИИу«ђтЇЋсђѓ

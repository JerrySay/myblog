# УДєжбЉТЋЎуеІТјеУЇљ

тЪ║уАђтЁЦжЌе№╝џ СИјMySQLуџёжЏХУиЮуд╗ТјЦУДд-ТЁЋУ»ЙуйЉ

Mysqlт╝ђтЈЉТіђтиД№╝џ MySQLт╝ђтЈЉТіђтиД№╝ѕСИђ№╝Ѕ MySQLт╝ђтЈЉТіђтиД№╝ѕС║ї№╝Ѕ MySQLт╝ђтЈЉТіђтиД№╝ѕСИЅ№╝Ѕ

Mysql5.7Тќ░уЅ╣ТђДтЈіуЏИтЁ│С╝ўтїќТіђтиД№╝џ MySQL5.7уЅѕТюгТќ░уЅ╣ТђД ТђДУЃйС╝ўтїќС╣ІMySQLС╝ўтїќ

MySQLжЏєуЙц№╝ѕPXC№╝ЅтЁЦжЌе MyCATтЁЦжЌетЈіт║ћуће

# тИИУДЂжЌ«жбўТђ╗у╗Њ

  • # РЉатГўтѓет╝ЋТЊј

    MySQLтИИУДЂуџёСИцуДЇтГўтѓет╝ЋТЊј№╝џMyISAMСИјInnoDBуџёуѕ▒ТЂеТЃЁС╗Є

  • # РЉАтГЌугджЏєтЈіТаАт»╣УДётѕЎ

    тГЌугджЏєТїЄуџёТў»СИђуДЇС╗јС║їУ┐ЏтѕХу╝ќуаЂтѕ░ТЪљу▒╗тГЌугдугдтЈиуџёТўат░ёсђѓТаАт»╣УДётѕЎтѕЎТў»ТїЄТЪљуДЇтГЌугджЏєСИІуџёТјњт║ЈУДётѕЎсђѓMysqlСИГТ»ЈСИђуДЇтГЌугджЏєжЃйС╝џт»╣т║ћСИђу│╗тѕЌуџёТаАт»╣УДётѕЎсђѓ

    MysqlжЄЄућеуџёТў»у▒╗С╝╝у╗ДТЅ┐уџёТќ╣т╝ЈТїЄт«џтГЌугджЏєуџёж╗ўУ«цтђ╝№╝їТ»ЈСИфТЋ░ТЇ«т║ЊС╗ЦтЈіТ»Јт╝аТЋ░ТЇ«УАежЃйТюЅУЄфти▒уџёж╗ўУ«цтђ╝№╝їС╗ќС╗гжђљт▒ѓу╗ДТЅ┐сђѓТ»ћтдѓ№╝џТЪљСИфт║ЊСИГТЅђТюЅУАеуџёж╗ўУ«цтГЌугджЏєт░єТў»У»ЦТЋ░ТЇ«т║ЊТЅђТїЄт«џуџётГЌугджЏє№╝ѕУ┐ЎС║ЏУАетюеТ▓АТюЅТїЄт«џтГЌугджЏєуџёТЃЁтєхСИІ№╝їТЅЇС╝џжЄЄућеж╗ўУ«цтГЌугджЏє№╝Ѕ PS№╝џТЋ┤уљєУЄфсђіJavaтиЦуеІтИѕС┐«уѓ╝С╣ІжЂЊсђІ

    У»ду╗єтєЁт«╣тЈ»С╗ЦтЈѓУђЃ№╝џ MySQLтГЌугджЏєтЈіТаАт»╣УДётѕЎуџёуљєУДБ

  • # РЉбу┤бт╝ЋуЏИтЁ│уџётєЁт«╣№╝ѕТЋ░ТЇ«т║ЊСй┐ућеСИГжЮътИИтЁ│жћ«уџёТіђТю»№╝їтљѕуљєТГБуА«уџёСй┐ућеу┤бт╝ЋтЈ»С╗ЦтцДтцДТЈљжФўТЋ░ТЇ«т║ЊуџёТЪЦУ»бТђДУЃй№╝Ѕ

    Mysqlу┤бт╝ЋСй┐ућеуџёТЋ░ТЇ«у╗ЊТъёСИ╗УдЂТюЅBTreeу┤бт╝Ћ тњї тЊѕтИїу┤бт╝Ћ сђѓт»╣С║јтЊѕтИїу┤бт╝ЋТЮЦУ»┤№╝їт║Ћт▒ѓуџёТЋ░ТЇ«у╗ЊТъёт░▒Тў»тЊѕтИїУАе№╝їтЏаТГцтюеу╗ЮтцДтцџТЋ░жюђТ▒ѓСИ║тЇЋТЮАУ«░тйЋТЪЦУ»буџёТЌХтђЎ№╝їтЈ»С╗ЦжђЅТІЕтЊѕтИїу┤бт╝Ћ№╝їТЪЦУ»бТђДУЃйТюђт┐Ф№╝ЏтЁХСйЎтцДжЃетѕєтю║ТЎ»№╝їт╗║У««жђЅТІЕBTreeу┤бт╝Ћсђѓ

    MysqlуџёBTreeу┤бт╝ЋСй┐ућеуџёТў»BТЋ░СИГуџёB+Tree№╝їСйєт»╣С║јСИ╗УдЂуџёСИцуДЇтГўтѓет╝ЋТЊјуџёт«ъуј░Тќ╣т╝ЈТў»СИЇтљїуџёсђѓ

    MyISAM: B+TreeтЈХУіѓуѓ╣уџёdataтЪЪтГўТћЙуџёТў»ТЋ░ТЇ«У«░тйЋуџётю░тЮђсђѓтюеу┤бт╝ЋТБђу┤буџёТЌХтђЎ№╝їждќтЁѕТїЅуЁДB+TreeТљюу┤бу«ЌТ│ЋТљюу┤бу┤бт╝Ћ№╝їтдѓТъюТїЄт«џуџёKeyтГўтюе№╝їтѕЎтЈќтЄ║тЁХ data тЪЪуџётђ╝№╝їуёХтљјС╗Ц data тЪЪуџётђ╝СИ║тю░тЮђУ»╗тЈќуЏИт║ћуџёТЋ░ТЇ«У«░тйЋсђѓУ┐ЎУбФуД░СИ║РђюжЮъУЂџу░Єу┤бт╝ЋРђЮсђѓ

    InnoDB: тЁХТЋ░ТЇ«ТќЄС╗ХТюгУ║Фт░▒Тў»у┤бт╝ЋТќЄС╗ХсђѓуЏИТ»ћMyISAM№╝їу┤бт╝ЋТќЄС╗ХтњїТЋ░ТЇ«ТќЄС╗ХТў»тѕєуд╗уџё№╝їтЁХУАеТЋ░ТЇ«ТќЄС╗ХТюгУ║Фт░▒Тў»ТїЅB+Treeу╗ёу╗ЄуџёСИђСИфу┤бт╝Ћу╗ЊТъё№╝їТаЉуџётЈХУіѓуѓ╣dataтЪЪС┐ЮтГўС║єт«їТЋ┤уџёТЋ░ТЇ«У«░тйЋсђѓУ┐ЎСИфу┤бт╝ЋуџёkeyТў»ТЋ░ТЇ«УАеуџёСИ╗жћ«№╝їтЏаТГцInnoDBУАеТЋ░ТЇ«ТќЄС╗ХТюгУ║Фт░▒Тў»СИ╗у┤бт╝ЋсђѓУ┐ЎУбФуД░СИ║РђюУЂџу░Єу┤бт╝Ћ№╝ѕТѕќУЂџжЏєу┤бт╝Ћ№╝ЅРђЮсђѓУђїтЁХСйЎуџёу┤бт╝ЋжЃйСйюСИ║УЙЁтіЕу┤бт╝Ћ№╝їУЙЁтіЕу┤бт╝ЋуџёdataтЪЪтГўтѓеуЏИт║ћУ«░тйЋСИ╗жћ«уџётђ╝УђїСИЇТў»тю░тЮђ№╝їУ┐ЎС╣ЪТў»тњїMyISAMСИЇтљїуџётю░Тќ╣сђѓтюеТа╣ТЇ«СИ╗у┤бт╝ЋТљюу┤бТЌХ№╝їуЏ┤ТјЦТЅЙтѕ░keyТЅђтюеуџёУіѓуѓ╣тЇ│тЈ»тЈќтЄ║ТЋ░ТЇ«№╝ЏтюеТа╣ТЇ«УЙЁтіЕу┤бт╝ЋТЪЦТЅЙТЌХ№╝їтѕЎжюђУдЂтЁѕтЈќтЄ║СИ╗жћ«уџётђ╝№╝їтюеУх░СИђжЂЇСИ╗у┤бт╝Ћсђѓ тЏаТГц№╝їтюеУ«ЙУ«АУАеуџёТЌХтђЎ№╝їСИЇт╗║У««Сй┐ућеУ┐ЄжЋ┐уџётГЌТ«хСйюСИ║СИ╗жћ«№╝їС╣ЪСИЇт╗║У««Сй┐ућежЮътЇЋУ░ЃуџётГЌТ«хСйюСИ║СИ╗жћ«№╝їУ┐ЎТаиС╝џжђаТѕљСИ╗у┤бт╝ЋжбЉу╣ЂтѕєУБѓсђѓ PS№╝џТЋ┤уљєУЄфсђіJavaтиЦуеІтИѕС┐«уѓ╝С╣ІжЂЊсђІ

    У»ду╗єтєЁт«╣тЈ»С╗ЦтЈѓУђЃ№╝џ

    т╣▓У┤Д№╝џmysqlу┤бт╝ЋуџёТЋ░ТЇ«у╗ЊТъё

    MySQLС╝ўтїќу│╗тѕЌ№╝ѕСИЅ№╝Ѕ--у┤бт╝ЋуџёСй┐ућесђЂтјЪуљєтњїУ«ЙУ«АС╝ўтїќ

    ТЋ░ТЇ«т║ЊСИцтцДуЦътЎесђљу┤бт╝ЋтњїжћЂсђЉ

  • # РЉБТЪЦУ»бу╝ЊтГўуџёСй┐уће

    my.cnfтіатЁЦС╗ЦСИІжЁЇуй«№╝їжЄЇтљ»Mysqlт╝ђтљ»ТЪЦУ»бу╝ЊтГў

    query_cache_type=1
query_cache_size=600000

    MysqlТЅДУАїС╗ЦСИІтЉйС╗цС╣ЪтЈ»С╗Цт╝ђтљ»ТЪЦУ»бу╝ЊтГў

    set global  query_cache_type=1;
set global  query_cache_size=600000;

    тдѓСИі№╝їт╝ђтљ»ТЪЦУ»бу╝ЊтГўтљјтюетљїТаиуџёТЪЦУ»бТЮАС╗ХС╗ЦтЈіТЋ░ТЇ«ТЃЁтєхСИІ№╝їС╝џуЏ┤ТјЦтюеу╝ЊтГўСИГУ┐ћтЏъу╗ЊТъюсђѓУ┐ЎжЄїуџёТЪЦУ»бТЮАС╗ХтїЁТІгТЪЦУ»бТюгУ║ФсђЂтйЊтЅЇУдЂТЪЦУ»буџёТЋ░ТЇ«т║ЊсђЂт«бТѕиуФ»тЇЈУ««уЅѕТюгтЈиуГЅСИђС║ЏтЈ»УЃйтй▒тЊЇу╗ЊТъюуџёС┐АТЂ»сђѓтЏаТГцС╗╗СйЋСИцСИфТЪЦУ»бтюеС╗╗СйЋтГЌугдСИіуџёСИЇтљїжЃйС╝џт»╝УЄ┤у╝ЊтГўСИЇтЉйСИГсђѓТГцтцќ№╝їтдѓТъюТЪЦУ»бСИГтїЁтљФС╗╗СйЋућеТѕиУЄфт«џС╣ЅтЄйТЋ░сђЂтГўтѓетЄйТЋ░сђЂућеТѕитЈўжЄЈсђЂСИ┤ТЌХУАесђЂMysqlт║ЊСИГуџёу│╗у╗ЪУАе№╝їтЁХТЪЦУ»бу╗ЊТъюС╣ЪСИЇС╝џУбФу╝ЊтГўсђѓ

    у╝ЊтГўт╗║уФІС╣Ітљј№╝їMysqlуџёТЪЦУ»бу╝ЊтГўу│╗у╗ЪС╝џУиЪУИфТЪЦУ»бСИГТХЅтЈіуџёТ»Јт╝аУАе№╝їтдѓТъюУ┐ЎС║ЏУАе№╝ѕТЋ░ТЇ«Тѕќу╗ЊТъё№╝ЅтЈЉућЪтЈўтїќ№╝їжѓБС╣ѕтњїУ┐Ўт╝аУАеуЏИтЁ│уџёТЅђТюЅу╝ЊтГўТЋ░ТЇ«жЃйт░єтц▒ТЋѕсђѓ

  • # РЉцС║ІтіАТю║тѕХ

  2. СИђУЄ┤ТђД№╝џ ТЅДУАїС║ІтіАтЅЇтљј№╝їТЋ░ТЇ«т║ЊС╗јСИђСИфСИђУЄ┤ТђДуіХТђЂУйгТЇбтѕ░тЈдСИђСИфСИђУЄ┤ТђДуіХТђЂсђѓ
  3. жџћуд╗ТђД№╝џ т╣ХтЈЉУ«┐жЌ«ТЋ░ТЇ«т║ЊТЌХ№╝їСИђСИфућеТѕиуџёС║ІуЅЕСИЇУбФтЁХС╗ќС║ІтіАТЅђт╣▓ТЅ░№╝їтљёт╣ХтЈЉС║ІтіАС╣ІжЌ┤ТЋ░ТЇ«т║ЊТў»уІгуФІуџё№╝Џ
  4. ТїЂС╣ЁТђД№╝џ СИђСИфС║ІтіАУбФТЈљС║цС╣Ітљјсђѓт«Ѓт»╣ТЋ░ТЇ«т║ЊСИГТЋ░ТЇ«уџёТћ╣тЈўТў»ТїЂС╣Ёуџё№╝їтЇ│Сй┐ТЋ░ТЇ«т║Њ тЈЉућЪТЋЁжџюС╣ЪСИЇт║ћУ»Цт»╣тЁХТюЅС╗╗СйЋтй▒тЊЇсђѓ

  • READ_UNCOMMITTED№╝ѕТюфТЈљС║цУ»╗№╝Ѕ: ТюђСйјуџёжџћуд╗у║ДтѕФ№╝їтЁЂУ«ИУ»╗тЈќт░џТюфТЈљС║цуџёТЋ░ТЇ«тЈўТЏ┤№╝їтЈ»УЃйС╝џт»╝УЄ┤УёЈУ»╗сђЂт╣╗У»╗ТѕќСИЇтЈ»жЄЇтцЇУ»╗

  • READ_COMMITTED№╝ѕТЈљС║цУ»╗№╝Ѕ: тЁЂУ«ИУ»╗тЈќт╣ХтЈЉС║ІтіАти▓у╗ЈТЈљС║цуџёТЋ░ТЇ«№╝їтЈ»С╗Цжў╗ТГбУёЈУ»╗№╝їСйєТў»т╣╗У»╗ТѕќСИЇтЈ»жЄЇтцЇУ»╗С╗ЇТюЅтЈ»УЃйтЈЉућЪ

  • REPEATABLE_READ№╝ѕтЈ»жЄЇтцЇУ»╗№╝Ѕ: т»╣тљїСИђтГЌТ«хуџётцџТгАУ»╗тЈќу╗ЊТъюжЃйТў»СИђУЄ┤уџё№╝їжЎцжЮъТЋ░ТЇ«Тў»УбФТюгУ║ФС║ІтіАУЄфти▒ТЅђС┐«Тћ╣№╝їтЈ»С╗Цжў╗ТГбУёЈУ»╗тњїСИЇтЈ»жЄЇтцЇУ»╗№╝їСйєт╣╗У»╗С╗ЇТюЅтЈ»УЃйтЈЉућЪсђѓ

    У┐ЎжЄїжюђУдЂТ│еТёЈуџёТў»№╝џMysql ж╗ўУ«цжЄЄућеуџё REPEATABLE_READжџћуд╗у║ДтѕФ Oracle ж╗ўУ«цжЄЄућеуџё READ_COMMITTEDжџћуд╗у║ДтѕФ.

    • УАїу║ДжћЂ№╝џ MysqlСИГжћЂт«џ у▓њт║дТюђт░Ј уџёСИђуДЇжћЂ№╝їтЈфжњѕт»╣тйЊтЅЇТЊЇСйюуџёУАїУ┐ЏУАїтіажћЂсђѓ УАїу║ДжћЂУЃйтцДтцДтЄЈт░ЉТЋ░ТЇ«т║ЊТЊЇСйюуџётє▓уфЂсђѓтЁХтіажћЂу▓њт║дТюђт░Ј№╝їт╣ХтЈЉт║джФў№╝їСйєтіажћЂуџёт╝ђжћђС╣ЪТюђтцД№╝їтіажћЂТЁб№╝їС╝џтЄ║уј░ТГ╗жћЂсђѓ

    • Gap lock№╝џжЌ┤жџЎжћЂ№╝їжћЂт«џСИђСИфУїЃтЏ┤№╝їСИЇтїЁТІгУ«░тйЋТюгУ║Ф

    тъѓуЏ┤ТІєтѕєуџёу╝║уѓ╣№╝џ СИ╗жћ«С╝џтЄ║уј░тєЌСйЎ№╝їжюђУдЂу«АуљєтєЌСйЎтѕЌ№╝їт╣ХС╝џт╝ЋУхиJoinТЊЇСйю№╝їтЈ»С╗ЦжђџУ┐Єтюет║ћућет▒ѓУ┐ЏУАїJoinТЮЦУДБтє│сђѓТГцтцќ№╝їтъѓуЏ┤тѕєтї║С╝џУ«ЕС║ІтіАтЈўтЙЌТЏ┤тіатцЇТЮѓ№╝Џ

  3. Т░┤т╣│тѕєтї║№╝џ

    С┐ЮТїЂТЋ░ТЇ«УАеу╗ЊТъёСИЇтЈў№╝їжђџУ┐ЄТЪљуДЇуГќуЋЦтГўтѓеТЋ░ТЇ«тѕєуЅЄсђѓУ┐ЎТаиТ»ЈСИђуЅЄТЋ░ТЇ«тѕєТЋБтѕ░СИЇтљїуџёУАеТѕќУђЁт║ЊСИГ№╝їУЙЙтѕ░С║єтѕєтИЃт╝ЈуџёуЏ«уџёсђѓ Т░┤т╣│ТІєтѕєтЈ»С╗ЦТћ»ТњЉжЮътИИтцДуџёТЋ░ТЇ«жЄЈсђѓ

    Т░┤т╣│ТІєтѕєТў»ТїЄТЋ░ТЇ«УАеУАїуџёТІєтѕє№╝їУАеуџёУАїТЋ░УХЁУ┐Є200СИЄУАїТЌХ№╝їт░▒С╝џтЈўТЁб№╝їУ┐ЎТЌХтЈ»С╗ЦТііСИђт╝ауџёУАеуџёТЋ░ТЇ«ТІєТѕљтцџт╝аУАеТЮЦтГўТћЙсђѓСИЙСИфСЙІтГљ№╝џТѕЉС╗гтЈ»С╗Цт░єућеТѕиС┐АТЂ»УАеТІєтѕєТѕљтцџСИфућеТѕиС┐АТЂ»УАе№╝їУ┐ЎТаит░▒тЈ»С╗ЦжЂ┐тЁЇтЇЋСИђУАеТЋ░ТЇ«жЄЈУ┐ЄтцДт»╣ТђДУЃйжђаТѕљтй▒тЊЇсђѓ

    Т░┤т╣│ТІєтѕєтЈ»С╗ЦТћ»ТїЂжЮътИИтцДуџёТЋ░ТЇ«жЄЈсђѓжюђУдЂТ│еТёЈуџёСИђуѓ╣Тў»:тѕєУАеС╗ЁС╗ЁТў»УДБтє│С║єтЇЋСИђУАеТЋ░ТЇ«У┐ЄтцДуџёжЌ«жбў№╝їСйєућ▒С║јУАеуџёТЋ░ТЇ«У┐ўТў»тюетљїСИђтЈ░Тю║тЎеСИі№╝їтЁХт«ът»╣С║јТЈљтЇЄMySQLт╣ХтЈЉУЃйтіЏТ▓АТюЅС╗ђС╣ѕТёЈС╣Ѕ№╝їТЅђС╗Ц Т░┤т╣│ТІєтѕєТюђтЦйтѕєт║Њ сђѓ

    Т░┤т╣│ТІєтѕєУЃйтцЪ Тћ»ТїЂжЮътИИтцДуџёТЋ░ТЇ«жЄЈтГўтѓе№╝їт║ћућеуФ»Тћ╣жђаС╣Ът░Љ№╝їСйє тѕєуЅЄС║ІтіАжџЙС╗ЦУДБтє│ №╝їУиеуЋїуѓ╣JoinТђДУЃйУЙЃти«№╝їжђ╗УЙЉтцЇТЮѓсђѓсђіJavaтиЦуеІтИѕС┐«уѓ╝С╣ІжЂЊсђІуџёСйюУђЁТјеУЇљ т░йжЄЈСИЇУдЂт»╣ТЋ░ТЇ«У┐ЏУАїтѕєуЅЄ№╝їтЏаСИ║ТІєтѕєС╝џтИдТЮЦжђ╗УЙЉсђЂжЃеуй▓сђЂУ┐љу╗┤уџётљёуДЇтцЇТЮѓт║д №╝їСИђУѕгуџёТЋ░ТЇ«УАетюеС╝ўтїќтЙЌтйЊуџёТЃЁтєхСИІТћ»ТњЉтЇЃСИЄС╗ЦСИІуџёТЋ░ТЇ«жЄЈТў»Т▓АТюЅтцфтцДжЌ«жбўуџёсђѓтдѓТъют«ътюеУдЂтѕєуЅЄ№╝їт░йжЄЈжђЅТІЕт«бТѕиуФ»тѕєуЅЄТъХТъё№╝їУ┐ЎТаитЈ»С╗ЦтЄЈт░ЉСИђТгАтњїСИГжЌ┤С╗ХуџёуйЉу╗юI/Oсђѓ

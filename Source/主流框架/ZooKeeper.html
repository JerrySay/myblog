# ZooKeeper уџёућ▒ТЮЦ

СИІжЮбУ┐ЎТ«хтєЁт«╣ТЉўУЄфсђіС╗јPaxosтѕ░Zookeeper сђІуггтЏЏуФауггСИђУіѓуџёТЪљТ«хтєЁт«╣№╝їТјеУЇљтцДт«ХжўЁУ»╗С╗ЦСИІ№╝џ

ZookeeperТюђТЌЕУхиТ║љС║јжЏЁУЎјуаћуЕХжЎбуџёСИђСИфуаћуЕХт░Ју╗ёсђѓтюетйЊТЌХ№╝їуаћуЕХС║║тЉўтЈЉуј░№╝їтюежЏЁУЎјтєЁжЃетЙѕтцџтцДтъІу│╗у╗ЪтЪ║ТюгжЃйжюђУдЂСЙЮУхќСИђСИфу▒╗С╝╝уџёу│╗у╗ЪТЮЦУ┐ЏУАїтѕєтИЃт╝ЈтЇЈУ░Ѓ№╝їСйєТў»У┐ЎС║Џу│╗у╗ЪтЙђтЙђжЃйтГўтюетѕєтИЃт╝ЈтЇЋуѓ╣жЌ«жбўсђѓТЅђС╗Ц№╝їжЏЁУЎјуџёт╝ђтЈЉС║║тЉўт░▒У»ЋтЏЙт╝ђтЈЉСИђСИфжђџућеуџёТЌатЇЋуѓ╣жЌ«жбўуџётѕєтИЃт╝ЈтЇЈУ░ЃТАєТъХ№╝їС╗ЦСЙ┐У«Ет╝ђтЈЉС║║тЉўт░єу▓ЙтіЏжЏєСИГтюетцёуљєСИџтіАжђ╗УЙЉСИісђѓ

тЁ│С║јРђюZooKeeperРђЮУ┐ЎСИфжА╣уЏ«уџётљЇтГЌ№╝їтЁХт«ъС╣ЪТюЅСИђТ«хУХБжЌ╗сђѓтюеуФІжА╣тѕЮТюЪ№╝їУђЃУЎЉтѕ░С╣ІтЅЇтєЁжЃетЙѕтцџжА╣уЏ«жЃйТў»Сй┐ућетіеуЅЕуџётљЇтГЌТЮЦтЉйтљЇуџё№╝ѕСЙІтдѓУЉЌтљЇуџёPigжА╣уЏ«),жЏЁУЎјуџётиЦуеІтИѕтИїТюЏу╗ЎУ┐ЎСИфжА╣уЏ«С╣ЪтЈќСИђСИфтіеуЅЕуџётљЇтГЌсђѓТЌХС╗╗уаћуЕХжЎбуџёждќтИГуДЉтГдт«ХRaghuRamakrishnanт╝ђујЕугЉтю░У»┤№╝џРђютюеУ┐ЎТаиСИІтј╗№╝їТѕЉС╗гУ┐Ўтё┐т░▒тЈўТѕљтіеуЅЕтЏГС║є№╝ЂРђЮТГцУ»ЮСИђтЄ║№╝їтцДт«Ху║иу║иУАеуц║т░▒тЈФтіеуЅЕтЏГу«АуљєтЉўтљДСИђСИђСИђтЏаСИ║тљёСИфС╗ЦтіеуЅЕтЉйтљЇуџётѕєтИЃт╝Ју╗ёС╗ХТћЙтюеСИђУхи№╝їжЏЁУЎјуџёТЋ┤СИфтѕєтИЃт╝Ју│╗у╗ЪуюІСИітј╗т░▒тЃЈСИђСИфтцДтъІуџётіеуЅЕтЏГС║є№╝їУђїZookeeperТГБтЦйУдЂућеТЮЦУ┐ЏУАїтѕєтИЃт╝Јуј»тбЃуџётЇЈУ░ЃСИђСИђС║јТў»№╝їZookeeperуџётљЇтГЌС╣Ът░▒ућ▒ТГцУ»ъућЪС║єсђѓ

# 1.1 ZooKeeper ТдѓУДѕ

ZooKeeper Тў»СИђСИфт╝ђТ║љуџётѕєтИЃт╝ЈтЇЈУ░ЃТюЇтіА№╝їZooKeeperТАєТъХТюђтѕЮТў»тюеРђюYahoo!"СИіТъёт╗║уџё№╝їућеС║јС╗Цу«ђтЇЋУђїуе│тЂЦуџёТќ╣т╝ЈУ«┐жЌ«С╗ќС╗гуџёт║ћућеуеІт║Јсђѓ тљјТЮЦ№╝їApache ZooKeeperТѕљСИ║Hadoop№╝їHBaseтњїтЁХС╗ќтѕєтИЃт╝ЈТАєТъХСй┐ућеуџёТюЅу╗ёу╗ЄТюЇтіАуџёТаЄтЄєсђѓ СЙІтдѓ№╝їApache HBaseСй┐ућеZooKeeperУиЪУИфтѕєтИЃт╝ЈТЋ░ТЇ«уџёуіХТђЂсђѓZooKeeper уџёУ«ЙУ«АуЏ«ТаЄТў»т░єжѓБС║ЏтцЇТЮѓСИћт«╣ТўЊтЄ║жћЎуџётѕєтИЃт╝ЈСИђУЄ┤ТђДТюЇтіАт░ЂУБЁУхиТЮЦ№╝їТъёТѕљСИђСИфжФўТЋѕтЈ»жЮауџётјЪУ»ГжЏє№╝їт╣ХС╗ЦСИђу│╗тѕЌу«ђтЇЋТўЊућеуџёТјЦтЈБТЈљСЙЏу╗ЎућеТѕиСй┐ућесђѓ

тјЪУ»Г№╝џ ТЊЇСйюу│╗у╗ЪТѕќУ«Ау«ЌТю║уйЉу╗юућеУ»ГУїЃуЋ┤сђѓТў»ућ▒УІЦт╣▓ТЮАТїЄС╗цу╗ёТѕљуџё№╝їућеС║јт«їТѕљСИђт«џтіЪУЃйуџёСИђСИфУ┐ЄуеІсђѓтЁиТюЅСИЇтЈ»тѕєтЅ▓ТђД┬итЇ│тјЪУ»ГуџёТЅДУАїт┐ЁжА╗Тў»У┐ъу╗Гуџё№╝їтюеТЅДУАїУ┐ЄуеІСИГСИЇтЁЂУ«ИУбФСИГТќГсђѓ

ZooKeeper Тў»СИђСИфтЁИтъІуџётѕєтИЃт╝ЈТЋ░ТЇ«СИђУЄ┤ТђДУДБтє│Тќ╣ТАѕ№╝їтѕєтИЃт╝Јт║ћућеуеІт║ЈтЈ»С╗ЦтЪ║С║ј ZooKeeper т«ъуј░У»ИтдѓТЋ░ТЇ«тЈЉтИЃ/У«бжўЁсђЂУ┤ЪУййтЮЄУААсђЂтЉйтљЇТюЇтіАсђЂтѕєтИЃт╝ЈтЇЈУ░Ѓ/жђџуЪЦсђЂжЏєуЙцу«АуљєсђЂMaster жђЅСИЙсђЂтѕєтИЃт╝ЈжћЂтњїтѕєтИЃт╝ЈжўЪтѕЌуГЅтіЪУЃйсђѓ

Zookeeper СИђСИфТюђтИИућеуџёСй┐ућетю║ТЎ»т░▒Тў»ућеС║јТІЁС╗╗ТюЇтіАућЪС║ДУђЁтњїТюЇтіАТХѕУ┤╣УђЁуџёТ│етєїСИГт┐Ѓ(ТЈљСЙЏтЈЉтИЃУ«бжўЁТюЇтіА)сђѓ ТюЇтіАућЪС║ДУђЁт░єУЄфти▒ТЈљСЙЏуџёТюЇтіАТ│етєїтѕ░ZookeeperСИГт┐Ѓ№╝їТюЇтіАуџёТХѕУ┤╣УђЁтюеУ┐ЏУАїТюЇтіАУ░ЃућеуџёТЌХтђЎтЁѕтѕ░ZookeeperСИГТЪЦТЅЙТюЇтіА№╝їУјитЈќтѕ░ТюЇтіАућЪС║ДУђЁуџёУ»ду╗єС┐АТЂ»С╣Ітљј№╝їтєЇтј╗У░ЃућеТюЇтіАућЪС║ДУђЁуџётєЁт«╣СИјТЋ░ТЇ«сђѓтдѓСИІтЏЙТЅђуц║№╝їтюе DubboТъХТъёСИГ Zookeeper т░▒ТІЁС╗╗С║єТ│етєїСИГт┐ЃУ┐ЎСИђУДњУЅ▓сђѓ

# 1.2 у╗ЊтљѕСИфС║║Сй┐ућеТЃЁтєхуџёУ«▓СИђСИІ ZooKeeper

тюеТѕЉУЄфти▒тЂџУ┐ЄуџёжА╣уЏ«СИГ№╝їСИ╗УдЂСй┐ућетѕ░С║є ZooKeeper СйюСИ║ Dubbo уџёТ│етєїСИГт┐Ѓ(Dubbo т«ўТќ╣ТјеУЇљСй┐уће ZooKeeperТ│етєїСИГт┐Ѓ)сђѓтЈдтцќтюеТљГт╗║ solr жЏєуЙцуџёТЌХтђЎ№╝їТѕЉСй┐уће ZooKeeper СйюСИ║ solr жЏєуЙцуџёу«АуљєтиЦтЁисђѓУ┐ЎТЌХ№╝їZooKeeper СИ╗УдЂТЈљСЙЏСИІжЮбтЄаСИфтіЪУЃй№╝џ1сђЂжЏєуЙцу«Ауљє№╝џт«╣жћЎсђЂУ┤ЪУййтЮЄУААсђѓ2сђЂжЁЇуй«ТќЄС╗ХуџёжЏєСИГу«Ауљє3сђЂжЏєуЙцуџётЁЦтЈБсђѓ

# С╗ђС╣ѕТў» RedLock

Redis т«ўТќ╣уФЎУ┐Ўу»ЄТќЄуФаТЈљтЄ║С║єСИђуДЇТЮЃтеЂуџётЪ║С║ј Redis т«ъуј░тѕєтИЃт╝ЈжћЂуџёТќ╣т╝ЈтљЇтЈФ Redlock№╝їТГцуДЇТќ╣т╝ЈТ»ћтјЪтЁѕуџётЇЋУіѓуѓ╣уџёТќ╣Т│ЋТЏ┤т«ЅтЁесђѓт«ЃтЈ»С╗ЦС┐ЮУ»ЂС╗ЦСИІуЅ╣ТђД№╝џ

  1. т«ЅтЁеуЅ╣ТђД№╝џС║њТќЦУ«┐жЌ«№╝їтЇ│Т░ИУ┐ютЈфТюЅСИђСИф client УЃйТІ┐тѕ░жћЂ
  2. жЂ┐тЁЇТГ╗жћЂ№╝џТюђу╗ѕ client жЃйтЈ»УЃйТІ┐тѕ░жћЂ№╝їСИЇС╝џтЄ║уј░ТГ╗жћЂуџёТЃЁтєх№╝їтЇ│Сй┐тјЪТюгжћЂСйЈТЪљУхёТ║љуџё client crash С║єТѕќУђЁтЄ║уј░С║єуйЉу╗ютѕєтї║
  3. т«╣жћЎТђД№╝џтЈфУдЂтцДжЃетѕє Redis Уіѓуѓ╣тГўТ┤╗т░▒тЈ»С╗ЦТГБтИИТЈљСЙЏТюЇтіА

# ТђјС╣ѕтюетЇЋУіѓуѓ╣СИіт«ъуј░тѕєтИЃт╝ЈжћЂ

# Redlock у«ЌТ│Ћ

у«ЌТ│ЋтЙѕТўЊТЄѓ№╝їУхи 5 СИф master Уіѓуѓ╣№╝їтѕєтИЃтюеСИЇтљїуџёТю║Тѕ┐т░йжЄЈС┐ЮУ»ЂтЈ»ућеТђДсђѓСИ║С║єУјитЙЌжћЂ№╝їclient С╝џУ┐ЏУАїтдѓСИІТЊЇСйю№╝џ

  1. тЙЌтѕ░тйЊтЅЇуџёТЌХжЌ┤№╝їтЙ«тдЎтЇЋСйЇ
  2. т░ЮУ»ЋжА║т║Јтю░тюе 5 СИфт«ъСЙІСИіућ│У»ижћЂ№╝їтйЊуёХжюђУдЂСй┐ућеуЏИтљїуџё key тњї random value№╝їУ┐ЎжЄїСИђСИф client жюђУдЂтљѕуљєУ«Йуй«СИј master Уіѓуѓ╣Т▓Ъжђџуџё timeout тцДт░Ј№╝їжЂ┐тЁЇжЋ┐ТЌХжЌ┤тњїСИђСИф fail С║єуџёУіѓуѓ╣ТхфУ┤╣ТЌХжЌ┤
  3. тйЊ client тюетцДС║јуГЅС║ј 3 СИф master СИіТѕљтіЪућ│У»итѕ░жћЂуџёТЌХтђЎ№╝їСИћт«ЃС╝џУ«Ау«Ќућ│У»ижћЂТХѕУђЌС║єтцџт░ЉТЌХжЌ┤№╝їУ┐ЎжЃетѕєТХѕУђЌуџёТЌХжЌ┤жЄЄућеУјитЙЌжћЂуџётйЊСИІТЌХжЌ┤тЄЈтј╗уггСИђТГЦУјитЙЌуџёТЌХжЌ┤Тѕ│тЙЌтѕ░№╝їтдѓТъюжћЂуџёТїЂу╗ГТЌХжЋ┐№╝ѕlock validity time№╝ЅТ»ћТхЂжђЮуџёТЌХжЌ┤тцџуџёУ»Ю№╝їжѓБС╣ѕжћЂт░▒уюЪТГБУјитЈќтѕ░С║єсђѓ
  4. тдѓТъюжћЂућ│У»итѕ░С║є№╝їжѓБС╣ѕжћЂуюЪТГБуџё lock validity time т║ћУ»ЦТў» origin№╝ѕlock validity time№╝Ѕ - ућ│У»ижћЂТюЪжЌ┤ТхЂжђЮуџёТЌХжЌ┤
  5. тдѓТъю client ућ│У»ижћЂтц▒У┤ЦС║є№╝їжѓБС╣ѕт«Ѓт░▒С╝џтюет░ЉжЃетѕєућ│У»иТѕљтіЪжћЂуџё master Уіѓуѓ╣СИіТЅДУАїжЄіТћЙжћЂуџёТЊЇСйю№╝їжЄЇуй«уіХТђЂ

# тц▒У┤ЦжЄЇУ»Ћ

тдѓТъюСИђСИф client ућ│У»ижћЂтц▒У┤ЦС║є№╝їжѓБС╣ѕт«ЃжюђУдЂуеЇуГЅСИђС╝џтюежЄЇУ»ЋжЂ┐тЁЇтцџСИф client тљїТЌХућ│У»ижћЂуџёТЃЁтєх№╝їТюђтЦйуџёТЃЁтєхТў»СИђСИф client жюђУдЂтЄаС╣јтљїТЌХтљЉ 5 СИф master тЈЉУхижћЂућ│У»исђѓтЈдтцќт░▒Тў»тдѓТъю client ућ│У»ижћЂтц▒У┤ЦС║єт«ЃжюђУдЂт░йт┐Фтюет«ЃТЏЙу╗Јућ│У»итѕ░жћЂуџё master СИіТЅДУАї unlock ТЊЇСйю№╝їСЙ┐С║јтЁХС╗ќ client УјитЙЌУ┐ЎТііжћЂ№╝їжЂ┐тЁЇУ┐ЎС║ЏжћЂУ┐ЄТюЪжђаТѕљуџёТЌХжЌ┤ТхфУ┤╣№╝їтйЊуёХтдѓТъюУ┐ЎТЌХтђЎуйЉу╗ютѕєтї║Сй┐тЙЌ client ТЌаТ│ЋУЂћу│╗СИіУ┐ЎС║Џ master№╝їжѓБС╣ѕУ┐ЎуДЇТхфУ┤╣т░▒Тў»СИЇтЙЌСИЇС╗ўтЄ║уџёС╗БС╗иС║єсђѓ

# 1 Atomic тјЪтГљу▒╗С╗Іу╗Ї

Atomic у┐╗У»ЉТѕљСИГТќЄТў»тјЪтГљуџёТёЈТђЮсђѓтюетїќтГдСИі№╝їТѕЉС╗гуЪЦжЂЊтјЪтГљТў»ТъёТѕљСИђУѕгуЅЕУ┤еуџёТюђт░ЈтЇЋСйЇ№╝їтюетїќтГдтЈЇт║ћСИГТў»СИЇтЈ»тѕєтЅ▓уџёсђѓтюеТѕЉС╗гУ┐ЎжЄї Atomic Тў»ТїЄСИђСИфТЊЇСйюТў»СИЇтЈ»СИГТќГуџёсђѓтЇ│Сй┐Тў»тюетцџСИфу║┐уеІСИђУхиТЅДУАїуџёТЌХтђЎ№╝їСИђСИфТЊЇСйюСИђТЌдт╝ђтДІ№╝їт░▒СИЇС╝џУбФтЁХС╗ќу║┐уеІт╣▓ТЅ░сђѓ

ТЅђС╗Ц№╝їТЅђУ░ЊтјЪтГљу▒╗У»┤у«ђтЇЋуѓ╣т░▒Тў»тЁиТюЅтјЪтГљ/тјЪтГљТЊЇСйюуЅ╣тЙЂуџёу▒╗сђѓ

т╣ХтЈЉтїЁ java.util.concurrent уџётјЪтГљу▒╗жЃйтГўТћЙтюеjava.util.concurrent.atomicСИІ,тдѓСИІтЏЙТЅђуц║сђѓ

Та╣ТЇ«ТЊЇСйюуџёТЋ░ТЇ«у▒╗тъІ№╝їтЈ»С╗Цт░єJUCтїЁСИГуџётјЪтГљу▒╗тѕєСИ║4у▒╗

public class AtomicIntegerTest {

	public static void main(String[] args) {

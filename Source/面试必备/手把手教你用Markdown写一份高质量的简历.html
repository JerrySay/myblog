# Markdown у«ђтјєТеАТЮ┐Таит╝ЈСИђУДѕ

тЈ»С╗ЦуюІтѕ░ТѕЉТііУЂћу│╗Тќ╣т╝ЈТћЙтюеуггСИђСйЇ№╝їтЏаСИ║тЁгтЈИСИђУѕгС╝џСИјСйаУЂћу│╗№╝їТЅђС╗ЦТііУЂћу│╗Тќ╣т╝ЈТћЙтюеуггСИђСйЇС╣ЪТў»СИ║С║єТќ╣СЙ┐УЂћу│╗УђЃУЎЉсђѓ

# СИ║С╗ђС╣ѕУдЂуће Markdown тєЎу«ђтјє№╝Ъ

Markdown У»ГТ│Ћу«ђтЇЋ№╝їТўЊС║јСИіТЅІсђѓСй┐ућеТГБуА«уџё Markdown У»ГУеђтєЎтЄ║ТЮЦуџёу«ђтјєСИЇУ«║Тў»тюеТјњуЅѕУ┐ўТў»Та╝т╝ЈСИіжЃйТ»ћУЙЃт╣▓тЄђ№╝їТўЊС║јжўЁУ»╗сђѓтЈдтцќ№╝їСй┐уће Markdown тєЎу«ђтјєС╣ЪС╝џу╗ЎжЮбУ»Ћт«ўСИђуДЇСйаТ»ћУЙЃСИЊСИџуџёТёЪУДЅсђѓ

жЎцС║єУ┐ЎС║Џ№╝їТѕЉУДЅтЙЌСй┐уће Markdown тєЎу«ђтјєтЈ»С╗ЦтЙѕТќ╣СЙ┐т░єтЁХСИјPDFсђЂHTMLсђЂPNGТа╝т╝ЈС╣ІжЌ┤УйгТЇбсђѓтљјжЮбТѕЉС╝џС╗Іу╗Їтѕ░УйгТЇбТќ╣Т│Ћ№╝їтЈфжюђУдЂСИђТЮАтЉйС╗цСйат░▒тЈ»С╗Цт«ъуј░ Markdown тѕ░ PDFсђЂHTML СИј PNGС╣ІжЌ┤уџёТЌау╝ЮтѕЄТЇбсђѓ

СИІжЮбуџёСИђС║ЏтєЁт«╣ТѕЉтюеС╣ІтЅЇуџёСИђу»ЄТќЄуФаСИГти▓у╗ЈТЈљтѕ░У┐Є№╝їУ┐ЎжЄїтєЇУ»┤СИђжЂЇ№╝їТюђтљјС╝џтѕєС║ФтдѓСйЋт«ъуј░Markdown тѕ░ PDFсђЂHTMLсђЂPNGТа╝т╝ЈС╣ІжЌ┤УйгТЇбсђѓ

# СИ║С╗ђС╣ѕУ»┤у«ђтјєтЙѕжЄЇУдЂ№╝Ъ

тЂЄтдѓСйаТў»уйЉућ│№╝їСйауџёу«ђтјєт┐ЁуёХС╝џу╗ЈУ┐ЄHRуџёуГЏжђЅ№╝їСИђт╝ау«ђтјєHRтЈ»УЃйС╣Ът░▒Уі▒У┤╣10уДњжњЪуюІСИђСИІ№╝їуёХтљјHRт░▒С╝џтє│т«џСйаУ┐ЎСИђтЁ│Тў»FailУ┐ўТў»Passсђѓ

тЂЄтдѓСйаТў»тєЁТје№╝їтдѓТъюСйауџёу«ђтјєТ▓АТюЅС╗ђС╣ѕС╝ўті┐уџёУ»Ю№╝їт░▒у«ЌТў»тєЁТјеСйауџёС║║тєЇућет┐Ѓ№╝їС╣ЪТЌаУЃйСИ║тіЏсђѓ

тЈдтцќ№╝їт░▒у«ЌСйажђџУ┐ЄС║єуГЏжђЅ№╝їтљјжЮбуџёжЮбУ»ЋСИГ№╝їжЮбУ»Ћт«ўС╣ЪС╝џТа╣ТЇ«Сйауџёу«ђтјєТЮЦтѕцТќГСйауЕХуФЪТў»тљдтђ╝тЙЌС╗ќУі▒У┤╣тЙѕтцџТЌХжЌ┤тј╗жЮбУ»Ћсђѓ

# тєЎу«ђтјєуџёСИцтцДТ│ЋтѕЎ

уЏ«тЅЇтєЎу«ђтјєуџёТќ╣т╝ЈТюЅСИцуДЇТЎ«жЂЇУбФУ«цтЈ»№╝їСИђуДЇТў» STAR№╝ї СИђуДЇТў» FABсђѓ

STARТ│ЋтѕЎ№╝ѕSituation Task Action Result№╝Ѕ№╝џ

  • Situation№╝џ С║ІТЃЁТў»тюеС╗ђС╣ѕТЃЁтєхСИІтЈЉућЪ№╝Џ
  • Task:№╝џ СйаТў»тдѓСйЋТўјуА«СйауџёС╗╗тіАуџё№╝Џ
  • Action№╝џ жњѕт»╣У┐ЎТаиуџёТЃЁтєхтѕєТъљ№╝їСйажЄЄућеС║єС╗ђС╣ѕУАїтіеТќ╣т╝Ј№╝Џ
  • Result№╝џ у╗ЊТъюТђјТаи№╝їтюеУ┐ЎТаиуџёТЃЁтєхСИІСйатГдС╣атѕ░С║єС╗ђС╣ѕсђѓ

FAB Т│ЋтѕЎ№╝ѕFeature Advantage Benefit№╝Ѕ№╝џ

  • Feature№╝џ Тў»С╗ђС╣ѕ№╝Џ
  • Advantage№╝џ Т»ћтѕФС║║тЦйтюетЊфС║Џтю░Тќ╣№╝Џ
  • Benefit№╝џ тдѓТъюжЏЄСйБСйа№╝їТІЏУЂўТќ╣С╝џтЙЌтѕ░С╗ђС╣ѕтЦйтцёсђѓ

# жА╣уЏ«у╗ЈтјєТђјС╣ѕтєЎ№╝Ъ

у«ђтјєСИіТюЅСИђСИцСИфжА╣уЏ«у╗ЈтјєтЙѕТГБтИИ№╝їСйєТў»уюЪТГБУЃйТііжА╣уЏ«у╗ЈтјєтЙѕтЦйуџёт▒Ћуц║у╗ЎжЮбУ»Ћт«ўуџёжЮътИИт░Љсђѓт»╣С║јжА╣уЏ«у╗ЈтјєтцДт«ХтЈ»С╗ЦУђЃУЎЉС╗јтдѓСИІтЄауѓ╣ТЮЦтєЎ№╝џ

  1. т»╣жА╣уЏ«ТЋ┤СйЊУ«ЙУ«АуџёСИђСИфТёЪтЈЌ
  2. тюеУ┐ЎСИфжА╣уЏ«СИГСйаУ┤ЪУ┤БС║єС╗ђС╣ѕсђЂтЂџС║єС╗ђС╣ѕсђЂТІЁС╗╗С║єС╗ђС╣ѕУДњУЅ▓
  3. С╗јУ┐ЎСИфжА╣уЏ«СИГСйатГдС╝џС║єжѓБС║ЏСИюУЦ┐№╝їСй┐ућетѕ░С║єжѓБС║ЏТіђТю»№╝їтГдС╝џС║єжѓБС║ЏТќ░ТіђТю»уџёСй┐уће
  4. тЈдтцќжА╣уЏ«ТЈЈУ┐░СИГ№╝їТюђтЦйтЈ»С╗ЦСйЊуј░УЄфти▒уџёу╗╝тљѕу┤аУ┤е№╝їТ»ћтдѓСйаТў»тдѓСйЋтЇЈУ░ЃжА╣уЏ«у╗ёТѕљтЉўтЇЈтљїт╝ђтЈЉуџёТѕќУђЁтюежЂЄтѕ░ТЪљСИђСИфТБўТЅІуџёжЌ«жбўуџёТЌХтђЎСйаТў»тдѓСйЋУДБтє│уџёсђѓ

# СИЊСИџТіђУЃйУ»ЦТђјС╣ѕтєЎ№╝Ъ

тЁѕжЌ«СИђСИІСйаУЄфти▒С╝џС╗ђС╣ѕ№╝їуёХтљјуюІуюІСйаТёЈтљЉуџётЁгтЈИжюђУдЂС╗ђС╣ѕсђѓСИђУѕгHRтЈ»УЃйт╣ХСИЇтцфТЄѓТіђТю»№╝їТЅђС╗ЦС╗ќтюеуГЏжђЅу«ђтјєуџёТЌХтђЎтЈ»УЃйт░▒уЏ»уЮђСйаСИЊСИџТіђУЃйуџётЁ│жћ«У»ЇТЮЦуюІсђѓт»╣С║јтЁгтЈИТюЅУдЂТ▒ѓУђїСйаСИЇС╝џуџёТіђУЃй№╝їСйатЈ»С╗ЦУі▒тЄатцЕТЌХжЌ┤тГдС╣аСИђСИІ№╝їуёХтљјтюеу«ђтјєСИітЈ»С╗ЦтєЎСИіУЄфти▒С║єУДБУ┐ЎСИфТіђУЃйсђѓТ»ћтдѓСйатЈ»С╗ЦУ┐ЎТаитєЎ№╝џ

  1. т░йжЄЈжЂ┐тЁЇСИ╗УДѓУАеУ┐░№╝їт░ЉСИђуѓ╣У»ГС╣ЅТеАу│іуџётйбт«╣У»Ї№╝їт░йжЄЈУдЂу«ђТ┤ЂТўјС║є№╝їжђ╗УЙЉу╗ЊТъёТИЁТЎ░сђѓ

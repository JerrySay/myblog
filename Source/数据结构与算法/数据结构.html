# Queue

# С╗ђС╣ѕТў»жўЪтѕЌ

жўЪтѕЌТў»ТЋ░ТЇ«у╗ЊТъёСИГТ»ћУЙЃжЄЇУдЂуџёСИђуДЇу▒╗тъІ№╝їт«ЃТћ»ТїЂ FIFO№╝їт░ЙжЃеТи╗тіасђЂтц┤жЃетѕажЎц№╝ѕтЁѕУ┐ЏжўЪтѕЌуџётЁЃу┤атЁѕтЄ║жўЪтѕЌ№╝Ѕ№╝їУиЪТѕЉС╗гућЪТ┤╗СИГуџёТјњжўЪу▒╗С╝╝сђѓ

# жўЪтѕЌуџёуДЇу▒╗

  • тЇЋжўЪтѕЌ№╝ѕтЇЋжўЪтѕЌт░▒Тў»тИИУДЂуџёжўЪтѕЌ, Т»ЈТгАТи╗тіатЁЃу┤аТЌХ№╝їжЃйТў»Ти╗тіатѕ░жўЪт░Й№╝їтГўтюеРђютЂЄТ║бтЄ║РђЮуџёжЌ«жбўС╣Ът░▒Тў»ТўјТўјТюЅСйЇуй«тЇ┤СИЇУЃйТи╗тіауџёТЃЁтєх№╝Ѕ
  • тЙфуј»жўЪтѕЌ№╝ѕжЂ┐тЁЇС║єРђютЂЄТ║бтЄ║РђЮуџёжЌ«жбў№╝Ѕ

# Java жЏєтљѕТАєТъХСИГуџёжўЪтѕЌ Queue

Java жЏєтљѕСИГуџё Queue у╗ДТЅ┐УЄф Collection ТјЦтЈБ №╝їDeque, LinkedList, PriorityQueue, BlockingQueue уГЅу▒╗жЃйт«ъуј░С║єт«Ѓсђѓ Queue ућеТЮЦтГўТћЙ уГЅтЙЁтцёуљєтЁЃу┤а уџёжЏєтљѕ№╝їУ┐ЎуДЇтю║ТЎ»СИђУѕгућеС║ју╝Њтє▓сђЂт╣ХтЈЉУ«┐жЌ«сђѓ жЎцС║єу╗ДТЅ┐ Collection ТјЦтЈБуџёСИђС║ЏТќ╣Т│Ћ№╝їQueue У┐ўТи╗тіаС║єжбЮтцќуџё Ти╗тіасђЂтѕажЎцсђЂТЪЦУ»бТЊЇСйюсђѓ

# ТјеУЇљТќЄуФа

  • Java жЏєтљѕТи▒тЁЦуљєУДБ№╝ѕ9№╝Ѕ№╝џQueue жўЪтѕЌ

# Set

# С╗ђС╣ѕТў» Set

Set у╗ДТЅ┐С║ј Collection ТјЦтЈБ№╝їТў»СИђСИфСИЇтЁЂУ«ИтЄ║уј░жЄЇтцЇтЁЃу┤а№╝їт╣ХСИћТЌат║ЈуџёжЏєтљѕ№╝їСИ╗УдЂ HashSet тњї TreeSet СИцтцДт«ъуј░у▒╗сђѓ

тюетѕцТќГжЄЇтцЇтЁЃу┤ауџёТЌХтђЎ№╝їSet жЏєтљѕС╝џУ░Ѓуће hashCode()тњї equal()Тќ╣Т│ЋТЮЦт«ъуј░сђѓ

# УАЦтЁЁ№╝џТюЅт║ЈжЏєтљѕСИјТЌат║ЈжЏєтљѕУ»┤Тўј

  • ТюЅт║ЈжЏєтљѕ№╝џжЏєтљѕжЄїуџётЁЃу┤атЈ»С╗ЦТа╣ТЇ« key Тѕќ index У«┐жЌ« (ListсђЂMap)
  • ТЌат║ЈжЏєтљѕ№╝џжЏєтљѕжЄїуџётЁЃу┤атЈфУЃйжЂЇтјєсђѓ№╝ѕSet№╝Ѕ

# HashSet тњї TreeSet т║Ћт▒ѓТЋ░ТЇ«у╗ЊТъё

HashSet Тў»тЊѕтИїУАеу╗ЊТъё№╝їСИ╗УдЂтѕЕуће HashMap уџё key ТЮЦтГўтѓетЁЃу┤а№╝їУ«Ау«ЌТЈњтЁЦтЁЃу┤ауџё hashCode ТЮЦУјитЈќтЁЃу┤атюежЏєтљѕСИГуџёСйЇуй«№╝Џ

TreeSet Тў»у║бж╗ЉТаЉу╗ЊТъё№╝їТ»ЈСИђСИфтЁЃу┤ажЃйТў»ТаЉСИГуџёСИђСИфУіѓуѓ╣№╝їТЈњтЁЦуџётЁЃу┤ажЃйС╝џУ┐ЏУАїТјњт║Ј№╝Џ

# ТјеУЇљТќЄуФа

  • JavaжЏєтљѕ--Set(тЪ║уАђ)

# List

# С╗ђС╣ѕТў»List

тюе List СИГ№╝їућеТѕитЈ»С╗Цу▓ЙуА«ТјДтѕХтѕЌУАеСИГТ»ЈСИфтЁЃу┤ауџёТЈњтЁЦСйЇуй«№╝їтЈдтцќућеТѕитЈ»С╗ЦжђџУ┐ЄТЋ┤ТЋ░у┤бт╝Ћ№╝ѕтѕЌУАеСИГуџёСйЇуй«№╝ЅУ«┐жЌ«тЁЃу┤а№╝їт╣ХТљюу┤бтѕЌУАеСИГуџётЁЃу┤асђѓ СИј Set СИЇтљї№╝їList жђџтИИтЁЂУ«ИжЄЇтцЇуџётЁЃу┤асђѓ тЈдтцќ List Тў»ТюЅт║ЈжЏєтљѕУђї Set Тў»ТЌат║ЈжЏєтљѕсђѓ

# ListуџётИИУДЂт«ъуј░у▒╗

ArrayList Тў»СИђСИфТЋ░у╗ёжўЪтѕЌ№╝їуЏИтйЊС║јтіеТђЂТЋ░у╗ёсђѓт«Ѓућ▒ТЋ░у╗ёт«ъуј░№╝їжџЈТю║У«┐жЌ«ТЋѕујЄжФў№╝їжџЈТю║ТЈњтЁЦсђЂжџЈТю║тѕажЎцТЋѕујЄСйјсђѓ

LinkedList Тў»СИђСИфтЈїтљЉжЊЙУАесђѓт«ЃС╣ЪтЈ»С╗ЦУбФтйЊСйютаєТаѕсђЂжўЪтѕЌТѕќтЈїуФ»жўЪтѕЌУ┐ЏУАїТЊЇСйюсђѓLinkedListжџЈТю║У«┐жЌ«ТЋѕујЄСйј№╝їСйєжџЈТю║ТЈњтЁЦсђЂжџЈТю║тѕажЎцТЋѕујЄжФўсђѓ

Vector Тў»уЪбжЄЈжўЪтѕЌ№╝їтњїArrayListСИђТаи№╝їт«ЃС╣ЪТў»СИђСИфтіеТђЂТЋ░у╗ё№╝їућ▒ТЋ░у╗ёт«ъуј░сђѓСйєТў»ArrayListТў»жЮъу║┐уеІт«ЅтЁеуџё№╝їУђїVectorТў»у║┐уеІт«ЅтЁеуџёсђѓ

Stack Тў»Таѕ№╝їт«Ѓу╗ДТЅ┐С║јVectorсђѓт«ЃуџёуЅ╣ТђДТў»№╝џтЁѕУ┐ЏтљјтЄ║(FILO, First In Last Out)сђѓуЏИтЁ│жўЁУ»╗№╝џjavaТЋ░ТЇ«у╗ЊТъёСИју«ЌТ│ЋС╣ІТаѕ№╝ѕStack№╝ЅУ«ЙУ«АСИјт«ъуј░

  • LinkedList Т║љуаЂтГдС╣а

  • # 1 С║їтЈЅТаЉ

    С║їтЈЅТаЉ№╝ѕуЎЙт║дуЎЙуДЉ№╝Ѕ

    (1)т«їтЁеС║їтЈЅТаЉРђћРђћУІЦУ«ЙС║їтЈЅТаЉуџёжФўт║дСИ║h№╝їжЎцугг h т▒ѓтцќ№╝їтЁХт«Ѓтљёт▒ѓ (1№йъh-1) уџёу╗Њуѓ╣ТЋ░жЃйУЙЙтѕ░ТюђтцДСИфТЋ░№╝їуггhт▒ѓТюЅтЈХтГљу╗Њуѓ╣№╝їт╣ХСИћтЈХтГљу╗Њуѓ╣жЃйТў»С╗јтидтѕ░тЈ│СЙЮТгАТјњтИЃ№╝їУ┐Ўт░▒Тў»т«їтЁеС║їтЈЅТаЉсђѓ

    (2)Т╗АС║їтЈЅТаЉРђћРђћжЎцС║єтЈХу╗Њуѓ╣тцќТ»ЈСИђСИфу╗Њуѓ╣жЃйТюЅтидтЈ│тГљтЈХСИћтЈХтГљу╗Њуѓ╣жЃйтцётюеТюђт║Ћт▒ѓуџёС║їтЈЅТаЉсђѓ

    (3)т╣│УААС║їтЈЅТаЉРђћРђћт╣│УААС║їтЈЅТаЉтЈѕУбФуД░СИ║AVLТаЉ№╝ѕтї║тѕФС║јAVLу«ЌТ│Ћ№╝Ѕ№╝їт«ЃТў»СИђТБхС║їтЈЅТјњт║ЈТаЉ№╝їСИћтЁиТюЅС╗ЦСИІТђДУ┤е№╝џт«ЃТў»СИђТБхуЕ║ТаЉТѕќт«ЃуџётидтЈ│СИцСИфтГљТаЉуџёжФўт║дти«уџёу╗Ют»╣тђ╝СИЇУХЁУ┐Є1№╝їт╣ХСИћтидтЈ│СИцСИфтГљТаЉжЃйТў»СИђТБхт╣│УААС║їтЈЅТаЉсђѓ

  • # 2 т«їтЁеС║їтЈЅТаЉ

    т«їтЁеС║їтЈЅТаЉ№╝ѕуЎЙт║дуЎЙуДЉ№╝Ѕ

    т«їтЁеС║їтЈЅТаЉ№╝џтЈХУіѓуѓ╣тЈфУЃйтЄ║уј░тюеТюђСИІт▒ѓтњїТгАСИІт▒ѓ№╝їт╣ХСИћТюђСИІжЮбСИђт▒ѓуџёу╗Њуѓ╣жЃйжЏєСИГтюеУ»Цт▒ѓТюђтидУЙ╣уџёУІЦт╣▓СйЇуй«уџёС║їтЈЅТаЉ

  • # 3 Т╗АС║їтЈЅТаЉ

    Т╗АС║їтЈЅТаЉ№╝ѕуЎЙт║дуЎЙуДЉ№╝їтЏйтєЁтцќуџёт«џС╣ЅСИЇтљї№╝Ѕ

    тЏйтєЁТЋЎуеІт«џС╣Ѕ№╝џСИђСИфС║їтЈЅТаЉ№╝їтдѓТъюТ»ЈСИђСИфт▒ѓуџёу╗Њуѓ╣ТЋ░жЃйУЙЙтѕ░ТюђтцДтђ╝№╝їтѕЎУ┐ЎСИфС║їтЈЅТаЉт░▒Тў»Т╗АС║їтЈЅТаЉсђѓС╣Ът░▒Тў»У»┤№╝їтдѓТъюСИђСИфС║їтЈЅТаЉуџёт▒ѓТЋ░СИ║K№╝їСИћу╗Њуѓ╣Тђ╗ТЋ░Тў»(2^k) -1 №╝їтѕЎт«Ѓт░▒Тў»Т╗АС║їтЈЅТаЉсђѓ

  • # тає

    ТЋ░ТЇ«у╗ЊТъёС╣Ітаєуџёт«џС╣Ѕ

    таєТў»тЁиТюЅС╗ЦСИІТђДУ┤еуџёт«їтЁеС║їтЈЅТаЉ№╝џТ»ЈСИфу╗Њуѓ╣уџётђ╝жЃйтцДС║јТѕќуГЅС║јтЁХтидтЈ│тГЕтГљу╗Њуѓ╣уџётђ╝№╝їуД░СИ║тцДжАХтає№╝ЏТѕќУђЁТ»ЈСИфу╗Њуѓ╣уџётђ╝жЃйт░ЈС║јТѕќуГЅС║јтЁХтидтЈ│тГЕтГљу╗Њуѓ╣уџётђ╝№╝їуД░СИ║т░ЈжАХтає

  • # 4 С║їтЈЅТЪЦТЅЙТаЉ№╝ѕBST№╝Ѕ

    ТхЁУ░ѕу«ЌТ│ЋтњїТЋ░ТЇ«у╗ЊТъё: СИЃ С║їтЈЅТЪЦТЅЙТаЉ

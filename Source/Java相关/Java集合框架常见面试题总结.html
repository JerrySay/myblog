# Arraylist СИј LinkedList тї║тѕФ

Arraylistт║Ћт▒ѓСй┐ућеуџёТў»ТЋ░у╗ё№╝ѕтГўУ»╗ТЋ░ТЇ«ТЋѕујЄжФў№╝їТЈњтЁЦтѕажЎцуЅ╣т«џСйЇуй«ТЋѕујЄСйј№╝Ѕ№╝їLinkedListт║Ћт▒ѓСй┐ућеуџёТў»тЈїтљЉтЙфуј»жЊЙУАеТЋ░ТЇ«у╗ЊТъё№╝ѕТЈњтЁЦ№╝їтѕажЎцТЋѕујЄуЅ╣тѕФжФў№╝ЅсђѓтГдУ┐ЄТЋ░ТЇ«у╗ЊТъёУ┐ЎжЌеУ»ЙтљјТѕЉС╗гт░▒уЪЦжЂЊжЄЄућежЊЙУАетГўтѓе№╝їТЈњтЁЦ№╝їтѕажЎцтЁЃу┤аТЌХжЌ┤тцЇТЮѓт║дСИЇтЈЌтЁЃу┤аСйЇуй«уџётй▒тЊЇ№╝їжЃйТў»У┐ЉС╝╝O№╝ѕ1№╝ЅУђїТЋ░у╗ёСИ║У┐ЉС╝╝O№╝ѕn№╝Ѕ№╝їтЏаТГцтйЊТЋ░ТЇ«уЅ╣тѕФтцџ№╝їУђїСИћу╗ЈтИИжюђУдЂТЈњтЁЦтѕажЎцтЁЃу┤аТЌХт╗║У««жђЅућеLinkedList.СИђУѕгуеІт║ЈтЈфућеArraylistт░▒тцЪућеС║є№╝їтЏаСИ║СИђУѕгТЋ░ТЇ«жЄЈжЃйСИЇС╝џУЏ«тцД№╝їArraylistТў»Сй┐ућеТюђтцџуџёжЏєтљѕу▒╗сђѓ

# HashSetтдѓСйЋТБђТЪЦжЄЇтцЇ

тйЊСйаТііт»╣У▒АтіатЁЦHashSetТЌХ№╝їHashSetС╝џтЁѕУ«Ау«Ќт»╣У▒Ауџёhashcodeтђ╝ТЮЦтѕцТќГт»╣У▒АтіатЁЦуџёСйЇуй«№╝їтљїТЌХС╣ЪС╝џСИјтЁХС╗ќтіатЁЦуџёт»╣У▒Ауџёhashcodeтђ╝СйюТ»ћУЙЃ№╝їтдѓТъюТ▓АТюЅуЏИугдуџёhashcode№╝їHashSetС╝џтЂЄУ«Йт»╣У▒АТ▓АТюЅжЄЇтцЇтЄ║уј░сђѓСйєТў»тдѓТъютЈЉуј░ТюЅуЏИтљїhashcodeтђ╝уџёт»╣У▒А№╝їУ┐ЎТЌХС╝џУ░Ѓућеequals№╝ѕ№╝ЅТќ╣Т│ЋТЮЦТБђТЪЦhashcodeуЏИуГЅуџёт»╣У▒АТў»тљдуюЪуџёуЏИтљїсђѓтдѓТъюСИцУђЁуЏИтљї№╝їHashSetт░▒СИЇС╝џУ«ЕтіатЁЦТЊЇСйюТѕљтіЪсђѓ№╝ѕТЉўУЄфТѕЉуџёJavaтљ»УњЎС╣дсђіHead fist javaсђІуггС║їуЅѕ№╝Ѕ

hashCode№╝ѕ№╝ЅСИјequals№╝ѕ№╝ЅуџёуЏИтЁ│УДёт«џ№╝џ

  1. тдѓТъюСИцСИфт»╣У▒АуЏИуГЅ№╝їтѕЎhashcodeСИђт«џС╣ЪТў»уЏИтљїуџё
  2. СИцСИфт»╣У▒АуЏИуГЅ,т»╣СИцСИфequalsТќ╣Т│ЋУ┐ћтЏъtrue
  3. СИцСИфт»╣У▒АТюЅуЏИтљїуџёhashcodeтђ╝№╝їт«ЃС╗гС╣ЪСИЇСИђт«џТў»уЏИуГЅуџё
  4. у╗╝СИі№╝їequalsТќ╣Т│ЋУбФУдєуЏќУ┐Є№╝їтѕЎhashCodeТќ╣Т│ЋС╣Ът┐ЁжА╗УбФУдєуЏќ
  5. hashCode()уџёж╗ўУ«цУАїСИ║Тў»т»╣таєСИіуџёт»╣У▒АС║ДућЪуІгуЅ╣тђ╝сђѓтдѓТъюТ▓АТюЅжЄЇтєЎhashCode()№╝їтѕЎУ»ЦclassуџёСИцСИфт»╣У▒АТЌаУ«║тдѓСйЋжЃйСИЇС╝џуЏИуГЅ№╝ѕтЇ│Сй┐У┐ЎСИцСИфт»╣У▒АТїЄтљЉуЏИтљїуџёТЋ░ТЇ«№╝Ѕсђѓ

==СИјequalsуџётї║тѕФ

  1. ==Тў»тѕцТќГСИцСИфтЈўжЄЈТѕќт«ъСЙІТў»СИЇТў»ТїЄтљЉтљїСИђСИфтєЁтГўуЕ║жЌ┤ equalsТў»тѕцТќГСИцСИфтЈўжЄЈТѕќт«ъСЙІТЅђТїЄтљЉуџётєЁтГўуЕ║жЌ┤уџётђ╝Тў»СИЇТў»уЏИтљї
  2. ==Тў»ТїЄт»╣тєЁтГўтю░тЮђУ┐ЏУАїТ»ћУЙЃ equals()Тў»т»╣тГЌугдСИ▓уџётєЁт«╣У┐ЏУАїТ»ћУЙЃ3.==ТїЄт╝ЋућеТў»тљдуЏИтљї equals()ТїЄуџёТў»тђ╝Тў»тљдуЏИтљї

# comparable тњї comparatorуџётї║тѕФ

  • comparableТјЦтЈБт«ъжЎЁСИіТў»тЄ║УЄфjava.langтїЁ т«ЃТюЅСИђСИф compareTo(Object obj)Тќ╣Т│ЋућеТЮЦТјњт║Ј
  • comparatorТјЦтЈБт«ъжЎЁСИіТў»тЄ║УЄф java.util тїЁт«ЃТюЅСИђСИфcompare(Object obj1, Object obj2)Тќ╣Т│ЋућеТЮЦТјњт║Ј

СИђУѕгТѕЉС╗гжюђУдЂт»╣СИђСИфжЏєтљѕСй┐ућеУЄфт«џС╣ЅТјњт║ЈТЌХ№╝їТѕЉС╗гт░▒УдЂжЄЇтєЎcompareToТќ╣Т│ЋТѕќcompareТќ╣Т│Ћ№╝їтйЊТѕЉС╗гжюђУдЂт»╣ТЪљСИђСИфжЏєтљѕт«ъуј░СИцуДЇТјњт║ЈТќ╣т╝Ј№╝їТ»ћтдѓСИђСИфsongт»╣У▒АСИГуџёТГїтљЇтњїТГїТЅІтљЇтѕєтѕФжЄЄућеСИђуДЇТјњт║ЈТќ╣Т│ЋуџёУ»Ю№╝їТѕЉС╗гтЈ»С╗ЦжЄЇтєЎcompareToТќ╣Т│ЋтњїСй┐ућеУЄфтѕХуџёComparatorТќ╣Т│ЋТѕќУђЁС╗ЦСИцСИфComparatorТЮЦт«ъуј░ТГїтљЇТјњт║ЈтњїТГїТўЪтљЇТјњт║Ј№╝їуггС║їуДЇС╗БУАеТѕЉС╗гтЈфУЃйСй┐ућеСИцСИфтЈѓТЋ░уЅѕуџёCollections.sort().

# static{}жЮЎТђЂС╗БуаЂтЮЌСИј{}жЮъжЮЎТђЂС╗БуаЂтЮЌ№╝ѕТъёжђаС╗БуаЂтЮЌ№╝Ѕ

уЏИтљїуѓ╣№╝џ жЃйТў»тюеJVMтіаУййу▒╗ТЌХСИћтюеТъёжђаТќ╣Т│ЋТЅДУАїС╣ІтЅЇТЅДУАї№╝їтюеу▒╗СИГжЃйтЈ»С╗Цт«џС╣ЅтцџСИф№╝їт«џС╣ЅтцџСИфТЌХТїЅт«џС╣ЅуџёжА║т║ЈТЅДУАї№╝їСИђУѕгтюеС╗БуаЂтЮЌСИГт»╣СИђС║ЏstaticтЈўжЄЈУ┐ЏУАїУхІтђ╝сђѓ

СИЇтљїуѓ╣№╝џ жЮЎТђЂС╗БуаЂтЮЌтюежЮъжЮЎТђЂС╗БуаЂтЮЌС╣ІтЅЇТЅДУАї(жЮЎТђЂС╗БуаЂтЮЌРђћжЮъжЮЎТђЂС╗БуаЂтЮЌРђћТъёжђаТќ╣Т│Ћ)сђѓжЮЎТђЂС╗БуаЂтЮЌтЈфтюеуггСИђТгАnewТЅДУАїСИђТгА№╝їС╣ІтљјСИЇтєЇТЅДУАї№╝їУђїжЮъжЮЎТђЂС╗БуаЂтЮЌтюеТ»ЈnewСИђТгАт░▒ТЅДУАїСИђТгАсђѓ жЮъжЮЎТђЂС╗БуаЂтЮЌтЈ»тюеТЎ«жђџТќ╣Т│ЋСИГт«џС╣Ѕ(СИЇУ┐ЄСйюућеСИЇтцД)№╝ЏУђїжЮЎТђЂС╗БуаЂтЮЌСИЇУАїсђѓ

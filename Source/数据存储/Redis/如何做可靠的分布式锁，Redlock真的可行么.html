т╝ђу»ЄСйюУђЁУ«цСИ║уј░тюе Redis жђљТИљУбФСй┐ућетѕ░ТЋ░ТЇ«у«АуљєжбєтЪЪ№╝їУ┐ЎСИфжбєтЪЪжюђУдЂТЏ┤т╝║уџёТЋ░ТЇ«СИђУЄ┤ТђДтњїУђљС╣ЁТђД№╝їУ┐ЎСй┐тЙЌС╗ќТёЪтѕ░ТІЁт┐Ѓ№╝їтЏаСИ║У┐ЎСИЇТў» Redis ТюђтѕЮУ«ЙУ«АуџётѕЮУАи№╝ѕС║Іт«ъСИіУ┐ЎС╣ЪТў»тЙѕтцџСИџуЋїуеІт║ЈтЉўуџёУ»»тї║№╝їУХіТЮЦУХіТіі Redis тйЊТѕљТЋ░ТЇ«т║ЊтюеСй┐уће№╝Ѕ№╝їтЁХСИГтЪ║С║ј Redis уџётѕєтИЃт╝ЈжћЂт░▒Тў»С╗цС║║ТІЁт┐ЃуџётЁХСИђсђѓ

Martin ТїЄтЄ║ждќтЁѕСйаУдЂТўјуА«СйаСИ║С╗ђС╣ѕСй┐ућетѕєтИЃт╝ЈжћЂ№╝їСИ║С║єТђДУЃйУ┐ўТў»ТГБуА«ТђД№╝ЪСИ║С║єтИ«Сйатї║тѕєУ┐ЎС║їУђЁ№╝їтюеУ┐ЎТііжћЂ fail С║єуџёТЌХтђЎСйатЈ»С╗ЦУ»бжЌ«УЄфти▒С╗ЦСИІжЌ«жбў№╝џ

  1. УдЂТђДУЃйуџё№╝џ ТІЦТюЅУ┐ЎТііжћЂСй┐тЙЌСйаСИЇС╝џжЄЇтцЇті│тіе№╝ѕСЙІтдѓСИђСИф job тЂџС║єСИцТгА№╝Ѕ№╝їтдѓТъюУ┐ЎТііжћЂ fail С║є№╝їСИцСИфУіѓуѓ╣тљїТЌХтЂџС║єУ┐ЎСИф Job№╝їжѓБС╣ѕУ┐ЎСИф Job тбътіаС║єСйауџёТѕљТюгсђѓ
  2. УдЂТГБуА«ТђДуџё№╝џ ТІЦТюЅжћЂтЈ»С╗Цжў▓ТГбт╣ХтЈЉТЊЇСйюТ▒АТЪЊСйауџёу│╗у╗ЪТѕќУђЁТЋ░ТЇ«№╝їтдѓТъюУ┐ЎТііжћЂ fail С║єСИцСИфУіѓуѓ╣тљїТЌХТЊЇСйюС║єСИђС╗йТЋ░ТЇ«№╝їу╗ЊТъютЈ»УЃйТў»ТЋ░ТЇ«СИЇСИђУЄ┤сђЂТЋ░ТЇ«СИбтц▒сђЂfile тє▓уфЂуГЅ№╝їС╝џт»╝УЄ┤СИЦжЄЇуџётљјТъюсђѓ

СИіУ┐░С║їУђЁжЃйТў»жюђТ▒ѓжћЂуџёТГБуА«тю║ТЎ»№╝їСйєТў»Сйат┐ЁжА╗ТИЁТЦџУЄфти▒Тў»тЏаСИ║С╗ђС╣ѕтјЪтЏажюђУдЂтѕєтИЃт╝ЈжћЂсђѓ

тдѓТъюСйатЈфТў»СИ║С║єТђДУЃй№╝їжѓБТ▓Ат┐ЁУдЂуће Redlock№╝їт«ЃТѕљТюгжФўСИћтцЇТЮѓ№╝їСйатЈфућеСИђСИф Redis т«ъСЙІС╣ЪтцЪС║є№╝їТюђтцџтіаСИфС╗јжў▓ТГбСИ╗ТїѓС║єсђѓтйЊуёХ№╝їСйаСй┐ућетЇЋУіѓуѓ╣уџё Redis жѓБС╣ѕТќГућхТѕќУђЁСИђС║ЏТЃЁтєхСИІ№╝їСйаС╝џСИбтц▒жћЂ№╝їСйєТў»СйауџёуЏ«уџётЈфТў»тіажђЪТђДУЃйСИћТќГућхУ┐ЎуДЇС║ІТЃЁСИЇС╝џу╗ЈтИИтЈЉућЪ№╝їУ┐Ўт╣ХСИЇТў»С╗ђС╣ѕтцДжЌ«жбўсђѓт╣ХСИћтдѓТъюСйаСй┐ућеС║єтЇЋУіѓуѓ╣ Redis№╝їжѓБС╣ѕтЙѕТўЙуёХСйаУ┐ЎСИфт║ћућежюђУдЂуџёжћЂу▓њт║дТў»тЙѕТеАу│іу▓Ќу│Ўуџё№╝їС╣ЪСИЇС╝џТў»С╗ђС╣ѕжЄЇУдЂуџёТюЇтіАсђѓ

# Сй┐ућеТЌХжЌ┤ТЮЦУДБтє│СИђУЄ┤ТђД

Redlock ТЌаТ│ЋС║ДућЪ fencing token ТЌЕУ»ЦТѕљСИ║тюежюђТ▒ѓТГБуА«ТђДуџётю║ТЎ»СИІт╝Ѓућет«Ѓуџёуљєућ▒№╝їСйєУ┐ўТюЅСИђС║Џтђ╝тЙЌУ«еУ«║уџётю░Тќ╣сђѓ

тГдТю»уЋїТюЅСИфУ»┤Т│Ћ№╝їу«ЌТ│Ћт»╣ТЌХжЌ┤СИЇтЂџтЂЄУ«Й№╝џтЏаСИ║У┐ЏуеІтЈ»УЃйpauseСИђТ«хТЌХжЌ┤сђЂТЋ░ТЇ«тїЁтЈ»УЃйтЏаСИ║уйЉу╗ют╗ХУ┐Ът╗Хтљјтѕ░УЙЙсђЂТЌХжњЪтЈ»УЃйТа╣Тюгт░▒Тў»жћЎуџёсђѓУђїтЈ»жЮауџёу«ЌТ│ЋСЙЮТЌДУдЂтюеСИіУ┐░тЂЄУ«ЙСИІтЂџТГБуА«уџёС║ІТЃЁсђѓ

т»╣С║ј failure detector ТЮЦУ»┤№╝їtimeout тЈфУЃйСйюСИ║уїюТхІТЪљСИфУіѓуѓ╣ fail уџёСЙЮТЇ«№╝їтЏаСИ║уйЉу╗ют╗ХУ┐ЪсђЂТюгтю░ТЌХжњЪСИЇТГБуА«уГЅтЁХС╗ќтјЪтЏауџёжЎљтѕХсђѓУђЃУЎЉтѕ░ Redis Сй┐уће gettimeofday№╝їУђїСИЇТў»тЇЋУ░ЃуџёТЌХжњЪ№╝їС╝џтЈЌтѕ░у│╗у╗ЪТЌХжЌ┤уџётй▒тЊЇ№╝їтЈ»УЃйС╝џуфЂуёХтЅЇУ┐ЏТѕќУђЁтљјжђђСИђТ«хТЌХжЌ┤№╝їУ┐ЎС╝џт»╝УЄ┤СИђСИф key ТЏ┤т┐ФТѕќТЏ┤ТЁбтю░У┐ЄТюЪсђѓ

тЈ»УДЂ№╝їRedlock СЙЮУхќС║јУ«ИтцџТЌХжЌ┤тЂЄУ«Й№╝їт«ЃтЂЄУ«ЙТЅђТюЅ Redis Уіѓуѓ╣жЃйУЃйт»╣тљїСИђСИф Key тюетЁХУ┐ЄТюЪтЅЇТїЂТюЅти«СИЇтцџуџёТЌХжЌ┤сђЂУиЪУ┐ЄТюЪТЌХжЌ┤уЏИТ»ћуйЉу╗ют╗ХУ┐ЪтЙѕт░ЈсђЂУиЪУ┐ЄТюЪТЌХжЌ┤уЏИТ»ћУ┐ЏуеІ pause тЙѕуЪГсђѓ

# ућеСИЇтЈ»жЮауџёТЌХжЌ┤ТЅЊуа┤ Redlock

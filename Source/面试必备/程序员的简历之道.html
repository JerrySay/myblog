# уеІт║ЈтЉўуџёу«ђтјєт░▒У»ЦУ┐ЎТаитєЎ

# 1 тЅЇУеђ

СИђС╗йтЦйуџёу«ђтјєтЈ»С╗ЦтюеТЋ┤СИфућ│У»ижЮбУ»ЋС╗ЦтЈіжЮбУ»ЋУ┐ЄуеІСИГУхитѕ░жЮътИИтЦйуџёСйюућесђѓ тюеСИЇтцИтцДУЄфти▒УЃйтіЏуџёТЃЁтєхСИІ№╝їтєЎтЄ║СИђС╗йтЦйуџёу«ђтјєС╣ЪТў»СИђжА╣тЙѕТБњуџёУЃйтіЏсђѓ

# 2 СИ║С╗ђС╣ѕУ»┤у«ђтјєтЙѕжЄЇУдЂ№╝Ъ

# 2.1 тЁѕС╗јжЮбУ»ЋтЅЇТЮЦУ»┤

тЂЄтдѓСйаТў»уйЉућ│№╝їСйауџёу«ђтјєт┐ЁуёХС╝џу╗ЈУ┐ЄHRуџёуГЏжђЅ№╝їСИђт╝ау«ђтјєHRтЈ»УЃйС╣Ът░▒Уі▒У┤╣10уДњжњЪуюІСИђСИІ№╝їуёХтљјHRт░▒С╝џтє│т«џСйаУ┐ЎСИђтЁ│Тў»FailУ┐ўТў»Passсђѓ

тЂЄтдѓСйаТў»тєЁТје№╝їтдѓТъюСйауџёу«ђтјєТ▓АТюЅС╗ђС╣ѕС╝ўті┐уџёУ»Ю№╝їт░▒у«ЌТў»тєЁТјеСйауџёС║║тєЇућет┐Ѓ№╝їС╣ЪТЌаУЃйСИ║тіЏсђѓ

тЈдтцќ№╝їт░▒у«ЌСйажђџУ┐ЄС║єуГЏжђЅ№╝їтљјжЮбуџёжЮбУ»ЋСИГ№╝їжЮбУ»Ћт«ўС╣ЪС╝џТа╣ТЇ«Сйауџёу«ђтјєТЮЦтѕцТќГСйауЕХуФЪТў»тљдтђ╝тЙЌС╗ќУі▒У┤╣тЙѕтцџТЌХжЌ┤тј╗жЮбУ»Ћсђѓ

ТЅђС╗Ц№╝їу«ђтјєт░▒тЃЈТў»ТѕЉС╗гуџёСИђСИфжЌежЮбСИђТаи№╝їт«ЃтюетЙѕтцДуеІт║дСИітє│т«џС║єСйаУЃйтљдУ┐ЏтЁЦтѕ░СИІСИђУй«уџёжЮбУ»ЋСИГсђѓ

# 2.2 тєЇС╗јжЮбУ»ЋСИГТЮЦУ»┤

ТѕЉтЈЉуј░тцДт«ХТ»ћУЙЃтќюТгбуюІжЮбу╗Ј №╝їУ┐Ўуѓ╣ТЌатЈ»тјџжЮъ№╝їСйєТў»тцДжЃетѕєжЮбу╗ЈжЃйТ▓АтЉіУ»ЅСйатЙѕтцџжЌ«жбўжЃйТў»тюеуЅ╣т«џТЮАС╗ХСИІТЅЇжЌ«уџёсђѓСИЙСИфу«ђтЇЋуџёСЙІтГљ№╝џСИђУѕгТЃЁтєхСИІСйауџёу«ђтјєСИіТ│еТўјСйаС╝џуџёСИюУЦ┐ТЅЇС╝џУбФжЌ«тѕ░№╝ѕJavaсђЂТЋ░ТЇ«у╗ЊТъёсђЂуйЉу╗юсђЂу«ЌТ│ЋУ┐ЎС║ЏтЪ║уАђТў»Т»ЈСИфС║║т┐ЁжЌ«уџё№╝Ѕ№╝їТ»ћтдѓтєЎС║єСйаС╝џ redis,жѓБжЮбУ»Ћт«ўт░▒тЙѕтцДТдѓујЄС╝џжЌ«Сйа redis уџёСИђС║ЏжЌ«жбўсђѓТ»ћтдѓ№╝џredisуџётИИУДЂТЋ░ТЇ«у▒╗тъІтЈіт║ћућетю║ТЎ»сђЂredisТў»тЇЋу║┐уеІСИ║С╗ђС╣ѕУ┐ўУ┐ЎС╣ѕт┐ФсђЂ redis тњї memcached уџётї║тѕФсђЂredis тєЁтГўТиўТ▒░Тю║тѕХуГЅуГЅсђѓ

ТЅђС╗Ц№╝їждќтЁѕ№╝їСйаУдЂТўјуА«уџёСИђуѓ╣Тў»№╝џСйаСИЇС╝џуџёСИюУЦ┐т░▒СИЇУдЂтєЎтюеу«ђтјєСИісђѓтЈдтцќ№╝їСйаУдЂУђЃУЎЉСйаУ»ЦтдѓСйЋТЅЇУЃйУ«ЕСйауџёС║«уѓ╣тюеу«ђтјєСИГтЄИТўЙтЄ║ТЮЦ№╝їТ»ћтдѓ№╝џСйатюеТЪљТЪљжА╣уЏ«тЂџС║єС╗ђС╣ѕС║ІТЃЁУДБтє│С║єС╗ђС╣ѕжЌ«жбў№╝ѕтЈфУдЂТюЅжА╣уЏ«т░▒СИђт«џТюЅУдЂУДБтє│уџёжЌ«жбў№╝ЅсђЂСйауџёТЪљСИђСИфжА╣уЏ«жЄїСй┐ућеС║єС╗ђС╣ѕТіђТю»тљјТЋ┤СйЊТђДУЃйтњїт╣ХтЈЉжЄЈТЈљтЇЄС║єтЙѕтцџуГЅуГЅсђѓ

жЮбУ»ЋтњїтиЦСйюТў»СИцтЏъС║І№╝їУЂфТўјуџёС║║С╝џТііжЮбУ»Ћт«ўтЙђУЄфти▒ТЊЁжЋ┐уџёжбєтЪЪжбє№╝їтЁХС╗ќС║║тѕЎУбФжЮбУ»Ћт«ўуЅхуЮђж╝╗тГљУх░сђѓУЎйУ»┤жЮбУ»ЋтњїтиЦСйюТў»СИцтЏъС║І№╝їСйєТў»СйаУдЂТЃ│УдЂУјитЙЌУЄфти▒Т╗АТёЈуџё offer №╝їСйаУЄфУ║Фуџёт«ътіЏт┐ЁжА╗УдЂт╝║сђѓ

# 3 СИІжЮбУ┐ЎтЄауѓ╣Сйат┐ЁжА╗уЪЦжЂЊ

  1. тцДжЃетѕєтЁгтЈИуџёHRжЃйУ»┤ТѕЉС╗гСИЇуюІжЄЇтГдтјє№╝ѕжфЌСйауџё№╝Ђ№╝Ѕ№╝їСйєТў»тдѓТъюСйауџётГдТаАСИЇтЄ║С╝ЌуџёУ»Ю№╝їтЙѕжџЙтюеСИђтаєу«ђтјєСИГУё▒жбќУђїтЄ║№╝їжЎцжЮъСйауџёу«ђтјєСИіТюЅуЅ╣тѕФуџёС║«уѓ╣№╝їТ»ћтдѓ№╝џТЪљТЪљтцДтјѓуџёт«ъС╣ау╗ЈтјєсђЂУјитЙЌС║єТЪљТЪљтцДУхЏуџётЦќуГЅуГЅсђѓ
  2. тцДжЃетѕєт║ћт▒іућЪТЅЙтиЦСйюуџёуАгС╝цТў»Т▓АТюЅтиЦСйюу╗ЈжфїТѕќт«ъС╣ау╗Јтјє№╝їТЅђС╗ЦтдѓТъюСйаТў»т║ћт▒іућЪт░▒СИЇУдЂжћЎУ┐ЄуДІТІЏтњїТўЦТІЏсђѓСИђТЌджћЎУ┐Є№╝їСйатљјжЮбт░▒ТъЂтцДтЈ»УЃйС╝џжЮбСИ┤уцЙТІЏ№╝їУ┐ЎСИфТЌХтђЎТ▓АТюЅтиЦСйюу╗ЈжфїуџёСйатЈ»УЃйт░▒С╝џжЮбСИ┤тљёуДЇуб░тБЂ№╝їт»╝УЄ┤ТЅЙСИЇтѕ░СИђСИфтЦйуџётиЦСйю
  3. тєЎтюеу«ђтјєСИіуџёСИюУЦ┐СИђт«џУдЂТЁјжЄЇ№╝їУ┐ЎТў»жЮбУ»Ћт«ўтцДжЄЈТЈљжЌ«уџётю░Тќ╣№╝Џ
  4. т░єУЄфти▒уџёжА╣уЏ«у╗Јтјєт«їуЙјуџёт▒Ћуц║тЄ║ТЮЦжЮътИИжЄЇУдЂсђѓ

# 4 т┐ЁжА╗С║єУДБуџёСИцтцДТ│ЋтѕЎ

РЉаSTARТ│ЋтѕЎ№╝ѕSituation Task Action Result№╝Ѕ№╝џ

  • Situation№╝џ С║ІТЃЁТў»тюеС╗ђС╣ѕТЃЁтєхСИІтЈЉућЪ№╝Џ
  • Task:№╝џ СйаТў»тдѓСйЋТўјуА«СйауџёС╗╗тіАуџё№╝Џ
  • Action№╝џ жњѕт»╣У┐ЎТаиуџёТЃЁтєхтѕєТъљ№╝їСйажЄЄућеС║єС╗ђС╣ѕУАїтіеТќ╣т╝Ј№╝Џ
  • Result№╝џ у╗ЊТъюТђјТаи№╝їтюеУ┐ЎТаиуџёТЃЁтєхСИІСйатГдС╣атѕ░С║єС╗ђС╣ѕсђѓ

у«ђУђїУеђС╣І№╝їSTARТ│ЋтѕЎ№╝їт░▒Тў»СИђуДЇУ«▓У┐░УЄфти▒ТЋЁС║ІуџёТќ╣т╝Ј№╝їТѕќУђЁУ»┤№╝їТў»СИђСИфТИЁТЎ░сђЂТЮАуљєуџёСйюТќЄТеАТЮ┐сђѓСИЇу«АТў»С╗ђС╣ѕ№╝їтљѕуљєуєЪу╗ЃУ┐љућеТГцТ│ЋтѕЎ№╝їтЈ»С╗ЦУй╗ТЮЙуџёт»╣жЮбУ»Ћт«ўТЈЈУ┐░С║ІуЅЕуџёжђ╗УЙЉТќ╣т╝Ј№╝їУАеуј░тЄ║УЄфти▒тѕєТъљжўљУ┐░жЌ«жбўуџёТИЁТЎ░ТђДсђЂТЮАуљєТђДтњїжђ╗УЙЉТђДсђѓ

СИІжЮбУ┐ЎТ«хтєЁт«╣ТЉўУЄфуЎЙт║дуЎЙуДЉ№╝їТѕЉУДЅтЙЌтєЎуџёжЮътИИСИЇжћЎ№╝џ

STARТ│ЋтѕЎ№╝ї500т╝║жЮбУ»ЋжбўтЏъуГћТЌХуџёТіђтиДТ│ЋтѕЎ№╝їтцЄтЈЌжЮбУ»ЋУђЁТѕљтіЪУђЁтњї500т╝║HRуџёТјет┤Єсђѓ ућ▒С║јУ┐ЎСИфТ│ЋтѕЎУбФт╣┐Т│Џт║ћућеС║јжЮбУ»ЋжЌ«жбўуџётЏъуГћ№╝їт░йу«АТѕЉС╗гУ┐ўтюетєЎу«ђтјєжўХТ«х№╝їСйєТў»№╝їтєЎу«ђтјєТЌХУЃйТііжЮбУ»ЋуџёжЌ«жбўт░▒ТЃ│тЦй№╝їС╝џСй┐УЄфти▒ТЏ┤тіаСИ╗тіетњїУЄфС┐А№╝їтЂџтѕ░у«ђтјє№╝їжЮбУ»ЋтЁ│УЂћТђД№╝їжђ╗УЙЉТђДт╝║№╝їСИЇУЄ│С║јтюеСИђСИфТюѕтљјтј╗жЮбУ»Ћ№╝їтЇ┤Тііу«ђтјєжЄїуџёСИюУЦ┐жЃйт┐ўТјЅС║є№╝ѕТЏ┤СйЋтєхТюЅС║ЏТюІтЈІС╝џуеЇтЙ«тцИтцДу«ђтјєтєЁт«╣№╝Ѕ тюеТѕЉС╗гтєЎу«ђтјєТЌХ№╝їТ»ЈСИфС║║жЃйУдЂтєЎСИіУЄфти▒уџётиЦСйюу╗Јтјє№╝їТ┤╗тіеу╗Јтјє№╝їТЃ│т┐ЁТ»ЈСИђСИфтљїтГд№╝їжЃйС╝џУхиуаЂУі▒СИітЇітцЕућџУЄ│ТЏ┤жЋ┐уџёТЌХжЌ┤тј╗Тљют»╗УёЉТхижЄїТЅђТюЅТюЅтЁ│уџёу╗Јтјє№╝їС║ЅтЈќТЅЙтЄ║ТюђтЦйуџёСИюУЦ┐тєЎтюеу«ђтјєСИісђѓ СйєТў»ТГцТЌХ№╝їТѕЉС╗гУдЂТ│еТёЈС║є№╝їу«ђтјєСИіуџёС╗╗СйЋСИђСИфС┐АТЂ»уѓ╣жЃйТюЅтЈ»УЃйТѕљСИ║ТЌЦтљјжЮбУ»ЋТЌХуџёжЄЇуѓ╣ТЈљжЌ«т»╣У▒А№╝їТЅђС╗ЦУ»┤№╝їСИЇУЃйтЈфу«АтєЎСИіУ«ЕУЄфти▒ТёЪУДЅТюђуЅЏуџёу╗Јтјєт░▒т«їС║ІС║є№╝їУдЂТЃ│тѕ░С╗ітљј№╝їтюежЮбУ»ЋСИГ№╝їСйаТЅђтєЎуџёу╗ЈтјєСИЄСИђУбФжЮбУ»Ћт«ўжЌ«тѕ░№╝їСйауюЪуџёУЃйтЏъуГћтЙЌТхЂтѕЕ№╝їжА║уЋЁ№╝їСИћУЃйжђџУ┐ЄУ┐ЎТ«ху╗Јтјє№╝їУ»ЂТўјУЄфти▒ТГБТў»жђѓтљѕУ┐ЎСИфУЂїСйЇуџёС║║тљЌ№╝Ъ

РЉАFAB Т│ЋтѕЎ№╝ѕFeature Advantage Benefit№╝Ѕ№╝џ

  • Feature№╝џ Тў»С╗ђС╣ѕ№╝Џ
  • Advantage№╝џ Т»ћтѕФС║║тЦйтюетЊфС║Џтю░Тќ╣№╝Џ
  • Benefit№╝џ тдѓТъюжЏЄСйБСйа№╝їТІЏУЂўТќ╣С╝џтЙЌтѕ░С╗ђС╣ѕтЦйтцёсђѓ

у«ђтЇЋТЮЦУ»┤№╝їУ┐ЎСИфТ│ЋтѕЎСИ╗УдЂТў»У«ЕСйауџёжЮбУ»Ћт«ўуЪЦжЂЊСйауџёС╝ўті┐сђЂТІЏС║єСйаС╣Ітљјт»╣тЁгтЈИТюЅС╗ђС╣ѕтИ«тіЕсђѓ

# 5 жА╣уЏ«у╗ЈтјєТђјС╣ѕтєЎ№╝Ъ

у«ђтјєСИіТюЅСИђСИцСИфжА╣уЏ«у╗ЈтјєтЙѕТГБтИИ№╝їСйєТў»уюЪТГБУЃйТііжА╣уЏ«у╗ЈтјєтЙѕтЦйуџёт▒Ћуц║у╗ЎжЮбУ»Ћт«ўуџёжЮътИИт░Љсђѓт»╣С║јжА╣уЏ«у╗ЈтјєтцДт«ХтЈ»С╗ЦУђЃУЎЉС╗јтдѓСИІтЄауѓ╣ТЮЦтєЎ№╝џ

  1. т»╣жА╣уЏ«ТЋ┤СйЊУ«ЙУ«АуџёСИђСИфТёЪтЈЌ
  2. тюеУ┐ЎСИфжА╣уЏ«СИГСйаУ┤ЪУ┤БС║єС╗ђС╣ѕсђЂтЂџС║єС╗ђС╣ѕсђЂТІЁС╗╗С║єС╗ђС╣ѕУДњУЅ▓
  3. С╗јУ┐ЎСИфжА╣уЏ«СИГСйатГдС╝џС║єжѓБС║ЏСИюУЦ┐№╝їСй┐ућетѕ░С║єжѓБС║ЏТіђТю»№╝їтГдС╝џС║єжѓБС║ЏТќ░ТіђТю»уџёСй┐уће
  4. тЈдтцќжА╣уЏ«ТЈЈУ┐░СИГ№╝їТюђтЦйтЈ»С╗ЦСйЊуј░УЄфти▒уџёу╗╝тљѕу┤аУ┤е№╝їТ»ћтдѓСйаТў»тдѓСйЋтЇЈУ░ЃжА╣уЏ«у╗ёТѕљтЉўтЇЈтљїт╝ђтЈЉуџёТѕќУђЁтюежЂЄтѕ░ТЪљСИђСИфТБўТЅІуџёжЌ«жбўуџёТЌХтђЎСйаТў»тдѓСйЋУДБтє│уџётЈѕТѕќУђЁУ»┤СйатюеУ┐ЎСИфжА╣уЏ«ућеС║єС╗ђС╣ѕТіђТю»т«ъуј░С║єС╗ђС╣ѕтіЪУЃйТ»ћтдѓ:ућеredisтЂџу╝ЊтГўТЈљжФўУ«┐жЌ«жђЪт║дтњїт╣ХтЈЉжЄЈсђЂСй┐ућеТХѕТЂ»жўЪтѕЌтЅіт│░тњїжЎЇТхЂуГЅуГЅсђѓ

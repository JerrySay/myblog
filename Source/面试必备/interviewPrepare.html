У┐ЎТў»сђљтцЄТѕўТўЦТІЏ/уДІТІЏу│╗тѕЌсђЉуџёуггС║їу»ЄТќЄуФа№╝їСИ╗УдЂТў»у«ђтЇЋтю░С╗Іу╗ЇтдѓСйЋтј╗тЄєтцЄжЮбУ»Ћсђѓ

СИЇУ«║Тў»ТаАТІЏУ┐ўТў»уцЙТІЏжЃйжЂ┐тЁЇСИЇС║єтљёуДЇжЮбУ»ЋсђЂугћУ»Ћ№╝їтдѓСйЋтј╗тЄєтцЄУ┐ЎС║ЏСИюУЦ┐т░▒ТўЙтЙЌТа╝тцќжЄЇУдЂсђѓСИЇУ«║Тў»угћУ»ЋУ┐ўТў»жЮбУ»ЋжЃйТў»ТюЅуФатЈ»тЙфуџё№╝їТѕЉУ┐ЎСИфРђюТюЅуФатЈ»тЙфРђЮУ»┤уџёТёЈТђЮтЈфТў»У»┤т║ћт»╣ТіђТю»жЮбУ»ЋТў»тЈ»С╗ЦТЈљтЅЇтЄєтцЄсђѓ ТѕЉтЁХт«ъуЅ╣тѕФСИЇтќюТгбжѓБуДЇСИ┤У┐ЉУђЃУ»Ћт░▒ТЈљтЅЇУЃїтЋіУ«░тЋітљёуДЇжбўуџёУАїСИ║№╝їжЮътИИтЈЇт»╣№╝ЂТѕЉУДЅтЙЌУ┐ЎуДЇТќ╣Т│ЋуЅ╣тѕФТъЂуФ»№╝їУђїСИћтюеуеЇТюЅСИђуѓ╣у╗ЈжфїуџёжЮбУ»Ћт«ўжЮбтЅЇТў»Та╣ТюгТ▓АТюЅућеуџёсђѓт╗║У««тцДт«ХУ┐ўТў»СИђТГЦСИђСИфУёџтЇ░УИЈУИЈт«ът«ътю░Ух░сђѓ

# 1 тдѓСйЋУјитЈќтцДтјѓжЮбУ»ЋТю║С╝џ№╝Ъ

тюеУ«▓тдѓСйЋУјитЈќтцДтјѓжЮбУ»ЋТю║С╝џС╣ІтЅЇ№╝їтЁѕТЮЦу╗ЎтцДт«ХуДЉТЎ«/т»╣Т»ћСИђСИІСИцСИфТаАТІЏжЮътИИтИИУДЂуџёТдѓт┐хРђћРђћТўЦТІЏтњїуДІТІЏсђѓ

  1. ТІЏУЂўС║║ТЋ░ №╝џуДІТІЏтцџС║јТўЦТІЏ №╝Џ
  2. ТІЏУЂўТЌХжЌ┤ №╝џ уДІТІЏСИђУѕг7ТюѕтидтЈ│т╝ђтДІ№╝їтцДТдѓСИђуЏ┤ТїЂу╗Гтѕ░10Тюѕт║ЋсђѓСйєТў»тцДтјѓ№╝ѕтдѓBAT№╝ЅжЃйС╝џТЌЕт╝ђтДІТЌЕу╗ЊТЮЪ№╝їТЅђС╗ЦСИђт«џУдЂТііТЈАтЦйТЌХжЌ┤сђѓТўЦТІЏТюђСй│ТЌХжЌ┤СИ║3Тюѕ№╝їТгАСй│ТЌХжЌ┤СИ║4Тюѕ№╝їУ┐ЏтЁЦ5ТюѕтЪ║Тюгт░▒СИЇС╝џтєЇТюЅТўЦТІЏС║є№╝ѕжЄЉСИЅжЊХтЏЏ№╝Ѕсђѓ
  3. т║ћУЂўжџЙт║д №╝џуДІТІЏуЋЦтцДС║јТўЦТІЏ№╝Џ
  4. ТІЏУЂўтЁгтЈИ№╝џ уДІТІЏТЋ░жЄЈтцџ№╝їУђїТўЦТІЏТЋ░жЄЈУЙЃт░Љ№╝їСИђУѕгСИ║уДІТІЏуџёУАЦтЁЁсђѓ

у╗╝СИі№╝їСИђУѕгТЮЦУ»┤№╝їуДІТІЏуџётљФжЄЉжЄЈТўјТўЙТў»жФўС║јТўЦТІЏуџёсђѓ

СИІжЮбТѕЉт░▒У»┤СИђСИІТѕЉУЄфти▒уЪЦжЂЊуџёСИђС║ЏТќ╣Т│Ћ№╝їСИЇУ┐Єт║ћУ»ЦС╣ЪТХхуЏќС║єтцДжЃетѕєУјитЈќжЮбУ»ЋТю║С╝џуџёТќ╣Т│Ћсђѓ

  1. тЁ│Т│етцДтјѓт«ўуйЉ№╝їжџЈТЌХТіЋжђњу«ђтјє№╝ѕУх░ТхЂуеІуџёуйЉућ│№╝Ѕ№╝Џ
  2. у║┐СИІтЈѓтіат«БУ«▓С╝џ№╝їуЏ┤ТјЦТіЋжђњу«ђтјє№╝Џ
  3. ТЅЙтѕ░тИѕтЁётИѕтДљ/У«цУ»єуџёС║║№╝їтИ«т┐ЎтєЁТје№╝ѕУЃйтцЪУ«ЕСйажЂ┐т╝ђуйЉућ│у«ђтјєуГЏжђЅ№╝їугћУ»ЋуГЏжђЅ№╝їУ┐ўТў»Тї║СИЇжћЎуџё№╝їСИЇУ┐ЄС╣ЪУ┐ўТў»жюђУдЂСйауџёу«ђтјєтцЪТБњ№╝Ѕ№╝Џ
  4. тЇџт«бтЈЉТќЄУбФуюІСИГ/GithubС╝ўуДђт╝ђТ║љжА╣уЏ«СйюУђЁ№╝їтцДтјѓтєЁжЃеС║║тЉўжѓђУ»иСйажЮбУ»Ћ№╝Џ
  5. Т▒ѓУЂїу▒╗уйЉуФЎТіЋжђњу«ђтјє№╝ѕСИЇТў»тцфТјеУЇљ№╝їжђѓтљѕТхиТіЋ№╝Ѕ№╝Џ

жЎцС║єУ┐ЎС║ЏТќ╣Т│Ћ№╝їТѕЉС╣ЪжЂЄтѕ░У┐ЄУ┐ЎТаиуџёу╗Јтјє№╝џТюЅС║ЏтцДтЁгтЈИуџёСИђС║ЏжЃежЌетЈ»УЃйТџѓТЌХТ▓АТІЏтцЪС║║№╝їуёХтљјтдѓТъюСйауџёС║▓ТѕџТѕќУђЁТюІтЈІтѕџтЦйтюеУ┐ЎСИфтЁгтЈИ№╝їУђїСйаТГБтЦйтЈѕтюет»╗Т▒ѓoffer№╝їжѓБС╣ѕжЮбУ»ЋТю║С╝џтЪ║ТюгСИіТў»ТюЅС║є№╝їУђїСИћУ┐ЎуДЇжЮбУ»ЋуџёжџЙт║дтЦйтЃЈСИђУѕгУ┐ўТЎ«жЂЇТ»ћтЁХС╗ќТГБУДёжЮбУ»ЋСйјтЙѕтцџсђѓ

# 2 жЮбУ»ЋтЅЇуџётЄєтцЄ

# 2.1 тЄєтцЄУЄфти▒уџёУЄфТѕЉС╗Іу╗Ї

С╗јHRжЮбсђЂТіђТю»жЮбтѕ░жФўу«АжЮб/жЃежЌеСИ╗у«АжЮб№╝їжЮбУ»Ћт«ўСИђУѕгС╝џУ«ЕСйатЁѕУЄфТѕЉС╗Іу╗ЇСИђСИІ№╝їТЅђС╗ЦтЦйтЦйтЄєтцЄУЄфти▒уџёУЄфТѕЉС╗Іу╗ЇуюЪуџёжЮътИИжЄЇУдЂсђѓуйЉСИіСИђУѕгт╗║У««уџёТў»тЄєтцЄтЦйСИцС╗йУЄфТѕЉС╗Іу╗Ї№╝џСИђС╗йт»╣hrУ»┤уџё№╝їСИ╗УдЂУ«▓УЃйуфЂтЄ║УЄфти▒уџёу╗Јтјє№╝їС╝џуџёу╝ќуеІТіђТю»СИђУ»ГтИдУ┐Є№╝ЏтЈдСИђС╗йт»╣ТіђТю»жЮбУ»Ћт«ўУ»┤уџё№╝їСИ╗УдЂУ«▓УЄфти▒С╝џуџёТіђТю»у╗єУіѓ№╝їжА╣уЏ«у╗Јжфї№╝їу╗ЈтјєжѓБС║Џт░▒СИђУ»ГтИдУ┐Єсђѓ

ТѕЉУ┐ЎжЄїу«ђтЇЋтѕєС║ФСИђСИІТѕЉУЄфти▒уџёУЄфТѕЉС╗Іу╗ЇуџёСИђСИфу«ђтЇЋуџёТеАТЮ┐тљД№╝џ

жЮбУ»Ћт«ў№╝їТѓетЦй№╝ЂТѕЉтЈФТЪљТЪљсђѓтцДтГдТЌХжЌ┤ТѕЉСИ╗УдЂтѕЕућеУ»ЙтцќТЌХжЌ┤тГдС╣аТЪљТЪљсђѓтюеТаАТюЪжЌ┤тЈѓСИјУ┐ЄСИђСИфТЪљТЪљу│╗у╗Ъуџёт╝ђтЈЉ№╝їтЈдтцќ№╝їУЄфти▒тГдС╣аУ┐ЄуеІСИГС╣ЪтєЎУ┐ЄтЙѕтцџу│╗у╗ЪТ»ћтдѓТЪљТЪљу│╗у╗ЪсђѓтюетГдС╣аС╣ІСйЎ№╝їТѕЉТ»ћУЙЃтќюТгбжђџУ┐ЄтЇџт«бТЋ┤уљєтѕєС║ФУЄфти▒ТЅђтГдуЪЦУ»єсђѓТѕЉуј░тюеТў»ТЪљТЪљуцЙтї║уџёУ«цУ»ЂСйюУђЁ№╝їтєЎУ┐ЄТЪљТЪљтЙѕСИЇжћЎуџёТќЄуФасђѓтЈдтцќ№╝їТѕЉУјитЙЌУ┐ЄТЪљТЪљтЦќ,ТѕЉуџёGithubСИіт╝ђТ║љуџёТЪљСИфжА╣уЏ«ти▓у╗ЈТюЅтцџт░ЉStarС║єсђѓ

# 2.2 тЁ│С║јуЮђУБЁ

уЕ┐УЦ┐УБЁсђЂТЅЊжбєтИдсђЂт░Јуџ«жъІ№╝ЪNO№╝ЂNO№╝ЂNO№╝ЂУ┐ЎТў»С║њУЂћуйЉтЁгтЈИжЮбУ»ЋтЈѕСИЇТў»тј╗Ух░у║бТ»»№╝їТЅђС╗ЦСйатЈфжюђУдЂуЕ┐уџёу«ђтЇЋтцДТќ╣т░▒тЦй№╝їСИЇжюђУдЂтцфТГБт╝Јсђѓ

# 2.3 жџЈУ║ФтИдСИіУЄфти▒уџёТѕљу╗ЕтЇЋтњїу«ђтјє

ТюЅуџётЁгтЈИтюежЮбУ»ЋтЅЇжЃйС╝џУ«ЕСйаС║цСИђС╗йТѕљу╗ЕтЇЋтњїу«ђтјєтйЊтЂџжЮбУ»ЋСИГуџётЈѓУђЃсђѓ

# 2.4 тдѓТъюжюђУдЂугћУ»Ћт░▒ТЈљтЅЇтѕиСИђС║ЏугћУ»Ћжбў

т╣│ТЌХуЕ║жЌ▓ТЌХжЌ┤тцџуџётЈ»С╗ЦтѕиСИђСИІугћУ»ЋжбўуЏ«№╝ѕуЅЏт«буйЉСИіТюЅтЙѕтцџ№╝ЅсђѓСйєТў»СИЇУдЂтЈфтѕижЮбУ»Ћжбў№╝їСИЇтіеТЅІcode№╝їуеІт║ЈтЉўСИЇТў»СИ║С║єУђЃУ»ЋУђїтГўтюеуџёсђѓ

# 2.5 Уі▒ТЌХжЌ┤СИђС║Џжђ╗УЙЉжбў

жЮбУ»ЋСИГтЈЉуј░ТюЅС║ЏтЁгтЈИжЃйТюЅжђ╗УЙЉжбўТхІУ»Ћуј»Уіѓ№╝їт╣ХСИћжЃйТііжђ╗УЙЉугћУ»ЋТѕљу╗ЕСйюСИ║тЙѕжЄЇУдЂуџёСИђСИфтЈѓУђЃсђѓ

# 2.6 тЄєтцЄтЦйУЄфти▒уџёжА╣уЏ«С╗Іу╗Ї

тдѓТъюТюЅжА╣уЏ«уџёУ»Ю№╝їТіђТю»жЮбУ»ЋуггСИђТГЦ№╝їжЮбУ»Ћт«ўСИђУѕгжЃйТў»У«ЕСйаУЄфти▒С╗Іу╗ЇСИђСИІСйауџёжА╣уЏ«сђѓСйатЈ»С╗ЦС╗јСИІжЮбтЄаСИфТќ╣тљЉТЮЦУђЃУЎЉ№╝џ

  1. т»╣жА╣уЏ«ТЋ┤СйЊУ«ЙУ«АуџёСИђСИфТёЪтЈЌ№╝ѕжЮбУ»Ћт«ўтЈ»УЃйС╝џУ«ЕСйаућ╗у│╗у╗ЪуџёТъХТъётЏЙ№╝Ѕ
  2. тюеУ┐ЎСИфжА╣уЏ«СИГСйаУ┤ЪУ┤БС║єС╗ђС╣ѕсђЂтЂџС║єС╗ђС╣ѕсђЂТІЁС╗╗С║єС╗ђС╣ѕУДњУЅ▓
  3. С╗јУ┐ЎСИфжА╣уЏ«СИГСйатГдС╝џС║єжѓБС║ЏСИюУЦ┐№╝їСй┐ућетѕ░С║єжѓБС║ЏТіђТю»№╝їтГдС╝џС║єжѓБС║ЏТќ░ТіђТю»уџёСй┐уће

# 2.7 ТЈљтЅЇтЄєтцЄТіђТю»жЮбУ»Ћ

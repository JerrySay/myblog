тюе Spring СИГ№╝їжѓБС║Џу╗ёТѕљт║ћућеуеІт║ЈуџёСИ╗СйЊтЈіућ▒ Spring IOC т«╣тЎеТЅђу«Ауљєуџёт»╣У▒А№╝їУбФуД░С╣ІСИ║ beanсђѓу«ђтЇЋтю░У«▓№╝їbean т░▒Тў»ућ▒ IOC т«╣тЎетѕЮтДІтїќсђЂУБЁжЁЇтЈіу«Ауљєуџёт»╣У▒А№╝їжЎцТГцС╣Ітцќ№╝їbean т░▒СИјт║ћућеуеІт║ЈСИГуџётЁХС╗ќт»╣У▒АТ▓АТюЅС╗ђС╣ѕтї║тѕФС║єсђѓУђї bean уџёт«џС╣ЅС╗ЦтЈі bean уЏИС║њжЌ┤уџёСЙЮУхќтЁ│у│╗т░єжђџУ┐ЄжЁЇуй«тЁЃТЋ░ТЇ«ТЮЦТЈЈУ┐░сђѓ

SpringСИГуџёbeanж╗ўУ«цжЃйТў»тЇЋСЙІуџё№╝їУ┐ЎС║ЏтЇЋСЙІBeanтюетцџу║┐уеІуеІт║ЈСИІтдѓСйЋС┐ЮУ»Ђу║┐уеІт«ЅтЁетЉб№╝Ъ СЙІтдѓт»╣С║јWebт║ћућеТЮЦУ»┤№╝їWebт«╣тЎет»╣С║јТ»ЈСИфућеТѕиУ»иТ▒ѓжЃйтѕЏт╗║СИђСИфтЇЋуІгуџёSevletу║┐уеІТЮЦтцёуљєУ»иТ▒ѓ№╝їт╝ЋтЁЦSpringТАєТъХС╣Ітљј№╝їТ»ЈСИфActionжЃйТў»тЇЋСЙІуџё№╝їжѓБС╣ѕт»╣С║јSpringТЅўу«АуџётЇЋСЙІService Bean№╝їтдѓСйЋС┐ЮУ»ЂтЁХт«ЅтЁетЉб№╝Ъ SpringуџётЇЋСЙІТў»тЪ║С║јBeanFactoryС╣Ът░▒Тў»Springт«╣тЎеуџё№╝їтЇЋСЙІBeanтюеТГцт«╣тЎетєЁтЈфТюЅСИђСИф№╝їJavaуџётЇЋСЙІТў»тЪ║С║ј JVM№╝їТ»ЈСИф JVM тєЁтЈфТюЅСИђСИфт«ъСЙІсђѓ

тюетцДтцџТЋ░ТЃЁтєхСИІсђѓтЇЋСЙІ bean Тў»тЙѕуљєТЃ│уџёТќ╣ТАѕсђѓСИЇУ┐Є№╝їТюЅТЌХтђЎСйатЈ»УЃйС╝џтЈЉуј░СйаТЅђСй┐ућеуџёу▒╗Тў»ТўЊтЈўуџё№╝їт«ЃС╗гС╝џС┐ЮТїЂСИђС║ЏуіХТђЂ№╝їтЏаТГцжЄЇућеТў»СИЇт«ЅтЁеуџёсђѓтюеУ┐ЎуДЇТЃЁтєхСИІ№╝їт░є class тБ░ТўјСИ║тЇЋСЙІуџёт░▒СИЇТў»жѓБС╣ѕТўјТЎ║С║єсђѓтЏаСИ║т»╣У▒АС╝џУбФТ▒АТЪЊ№╝їуеЇтљјжЄЇућеуџёТЌХтђЎС╝џтЄ║уј░ТёЈТЃ│СИЇтѕ░уџёжЌ«жбўсђѓТЅђС╗Ц Spring т«џС╣ЅС║єтцџуДЇСйюућетЪЪуџёbeanсђѓ

# СИђ beanуџёСйюућетЪЪ

тѕЏт╗║СИђСИфbeanт«џС╣Ѕ№╝їтЁХт«ъУ┤еТў»ућеУ»Цbeanт«џС╣Ѕт»╣т║ћуџёу▒╗ТЮЦтѕЏт╗║уюЪТГБт«ъСЙІуџёРђюжЁЇТќ╣РђЮсђѓТііbeanт«џС╣ЅуюІТѕљСИђСИфжЁЇТќ╣тЙѕТюЅТёЈС╣Ѕ№╝їт«ЃСИјclassтЙѕу▒╗С╝╝№╝їтЈфТа╣ТЇ«СИђт╝аРђютцёТќ╣РђЮт░▒тЈ»С╗ЦтѕЏт╗║тцџСИфт«ъСЙІсђѓСИЇС╗ЁтЈ»С╗ЦТјДтѕХТ│етЁЦтѕ░т»╣У▒АСИГуџётљёуДЇСЙЮУхќтњїжЁЇуй«тђ╝№╝їУ┐ўтЈ»С╗ЦТјДтѕХУ»Цт»╣У▒АуџёСйюућетЪЪсђѓУ┐ЎТаитЈ»С╗ЦуЂхТ┤╗жђЅТІЕТЅђт╗║т»╣У▒АуџёСйюућетЪЪ№╝їУђїСИЇт┐ЁтюеJava Classу║Дт«џС╣ЅСйюућетЪЪсђѓSpring FrameworkТћ»ТїЂС║ћуДЇСйюућетЪЪ№╝їтѕєтѕФжўљУ┐░тдѓСИІУАесђѓ

С║ћуДЇСйюућетЪЪСИГ№╝їrequestсђЂsession тњї global session СИЅуДЇСйюућетЪЪС╗ЁтюетЪ║С║јwebуџёт║ћућеСИГСй┐уће№╝ѕСИЇт┐ЁтЁ│т┐ЃСйаТЅђжЄЄућеуџёТў»С╗ђС╣ѕwebт║ћућеТАєТъХ№╝Ѕ№╝їтЈфУЃйућетюетЪ║С║ј web уџё Spring ApplicationContext уј»тбЃсђѓ

# 1. singletonРђћРђћтћ»СИђ bean т«ъСЙІ

тйЊСИђСИф bean уџёСйюућетЪЪСИ║ singleton№╝їжѓБС╣ѕSpring IoCт«╣тЎеСИГтЈфС╝џтГўтюеСИђСИфтЁ▒С║Фуџё bean т«ъСЙІ№╝їт╣ХСИћТЅђТюЅт»╣ bean уџёУ»иТ▒ѓ№╝їтЈфУдЂ id СИјУ»Ц bean т«џС╣ЅуЏИтї╣жЁЇ№╝їтѕЎтЈфС╝џУ┐ћтЏъbeanуџётљїСИђт«ъСЙІсђѓ singleton Тў»тЇЋСЙІу▒╗тъІ(т»╣т║ћС║јтЇЋСЙІТеАт╝Ј)№╝їт░▒Тў»тюетѕЏт╗║Ухит«╣тЎеТЌХт░▒тљїТЌХУЄфтіетѕЏт╗║С║єСИђСИфbeanуџёт»╣У▒А№╝їСИЇу«АСйаТў»тљдСй┐уће№╝їСйєТѕЉС╗гтЈ»С╗ЦТїЄт«џBeanУіѓуѓ╣уџё lazy-init=РђЮtrueРђЮ ТЮЦт╗ХУ┐ЪтѕЮтДІтїќbean№╝їУ┐ЎТЌХтђЎ№╝їтЈфТюЅтюеуггСИђТгАУјитЈќbeanТЌХТЅЇС╝џтѕЮтДІтїќbean№╝їтЇ│уггСИђТгАУ»иТ▒ѓУ»ЦbeanТЌХТЅЇтѕЮтДІтїќсђѓ Т»ЈТгАУјитЈќтѕ░уџёт»╣У▒АжЃйТў»тљїСИђСИфт»╣У▒АсђѓТ│еТёЈ№╝їsingleton СйюућетЪЪТў»SpringСИГуџёу╝║уюЂСйюућетЪЪсђѓУдЂтюеXMLСИГт░є bean т«џС╣ЅТѕљ singleton №╝їтЈ»С╗ЦУ┐ЎТаижЁЇуй«№╝џ

<bean id="ServiceImpl" class="cn.csdn.service.ServiceImpl" scope="singleton">

С╣ЪтЈ»С╗ЦжђџУ┐Є @Scope Т│еУДБ№╝ѕт«ЃтЈ»С╗ЦТўЙуц║ТїЄт«џbeanуџёСйюућеУїЃтЏ┤сђѓ№╝ЅуџёТќ╣т╝Ј

@Scope("singleton")
public class ServiceImpl{

}

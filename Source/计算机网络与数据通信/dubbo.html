# 1.1 С╗ђС╣ѕТў» Dubbo?

Apache Dubbo (incubating) |╦ѕd╩їb╔Ў╩і| Тў»СИђТгЙжФўТђДУЃйсђЂУй╗жЄЈу║Дуџёт╝ђТ║љJava RPC ТАєТъХ№╝їт«ЃТЈљСЙЏС║єСИЅтцДТаИт┐ЃУЃйтіЏ№╝џжЮбтљЉТјЦтЈБуџёУ┐юуеІТќ╣Т│ЋУ░Ѓуће№╝їТЎ║УЃйт«╣жћЎтњїУ┤ЪУййтЮЄУАА№╝їС╗ЦтЈіТюЇтіАУЄфтіеТ│етєїтњїтЈЉуј░сђѓу«ђтЇЋТЮЦУ»┤ Dubbo Тў»СИђСИфтѕєтИЃт╝ЈТюЇтіАТАєТъХ№╝їУЄ┤тіЏС║јТЈљСЙЏжФўТђДУЃйтњїжђЈТўјтїќуџёRPCУ┐юуеІТюЇтіАУ░ЃућеТќ╣ТАѕ№╝їС╗ЦтЈіSOAТюЇтіАТ▓╗уљєТќ╣ТАѕсђѓ

Dubbo уЏ«тЅЇти▓у╗ЈТюЅТјЦУ┐Љ 23k уџё Star №╝їDubboуџёGithub тю░тЮђ№╝џhttps://github.com/apache/incubator-dubbo сђѓ тЈдтцќ№╝їтюет╝ђТ║љСИГтЏйСИЙУАїуџё2018т╣┤т║дТюђтЈЌТгбУ┐јСИГтЏйт╝ђТ║љУй»С╗ХУ┐ЎСИфТ┤╗тіеуџёУ»ёжђЅСИГ№╝їDubbo ТЏ┤Тў»тЄГтђЪтЁХУХЁжФўС║║Т░ћС╗ЁТгАС║ј vue.js тњї ECharts УјитЙЌуггСИЅтљЇуџётЦйТѕљу╗Есђѓ

# 1.3 СИ║С╗ђС╣ѕУдЂуће Dubbo?

Dubbo уџёУ»ъућЪтњї SOA тѕєтИЃт╝ЈТъХТъёуџёТхЂУАїТюЅуЮђУјФтцДуџётЁ│у│╗сђѓSOA жЮбтљЉТюЇтіАуџёТъХТъё№╝ѕService Oriented Architecture№╝Ѕ№╝їС╣Ът░▒Тў»ТіітиЦуеІТїЅуЁДСИџтіАжђ╗УЙЉТІєтѕєТѕљТюЇтіАт▒ѓсђЂУАеуј░т▒ѓСИцСИфтиЦуеІсђѓТюЇтіАт▒ѓСИГтїЁтљФСИџтіАжђ╗УЙЉ№╝їтЈфжюђУдЂт»╣тцќТЈљСЙЏТюЇтіАтЇ│тЈ»сђѓУАеуј░т▒ѓтЈфжюђУдЂтцёуљєтњїжАхжЮбуџёС║цС║њ№╝їСИџтіАжђ╗УЙЉжЃйТў»У░ЃућеТюЇтіАт▒ѓуџёТюЇтіАТЮЦт«ъуј░сђѓSOAТъХТъёСИГТюЅСИцСИфСИ╗УдЂУДњУЅ▓№╝џТюЇтіАТЈљСЙЏУђЁ№╝ѕProvider№╝ЅтњїТюЇтіАСй┐ућеУђЁ№╝ѕConsumer№╝Ѕсђѓ

тдѓТъюСйаУдЂт╝ђтЈЉтѕєтИЃт╝ЈуеІт║Ј№╝їСйаС╣ЪтЈ»С╗ЦуЏ┤ТјЦтЪ║С║ј HTTP ТјЦтЈБУ┐ЏУАїжђџС┐А№╝їСйєТў»СИ║С╗ђС╣ѕУдЂуће DubboтЉб№╝Ъ

  3. ТюЇтіАУ«┐жЌ«тјІтіЏС╗ЦтЈіТЌХжЋ┐у╗ЪУ«АсђЂУхёТ║љУ░Ѓт║дтњїТ▓╗уљєРђћРђћтЪ║С║јУ«┐жЌ«тјІтіЏт«ъТЌХу«АуљєжЏєуЙцт«╣жЄЈ№╝їТЈљжФўжЏєуЙцтѕЕућеујЄсђѓ
  4. ТюЇтіАжЎЇу║ДРђћРђћТЪљСИфТюЇтіАТїѓТјЅС╣ІтљјУ░ЃућетцЄућеТюЇтіА

тЈдтцќ№╝їDubbo жЎцС║єУЃйтцЪт║ћућетюетѕєтИЃт╝Ју│╗у╗ЪСИГ№╝їС╣ЪтЈ»С╗Цт║ћућетюеуј░тюеТ»ћУЙЃуЂФуџётЙ«ТюЇтіАу│╗у╗ЪСИГсђѓСИЇУ┐Є№╝їућ▒С║ј Spring Cloud тюетЙ«ТюЇтіАСИГт║ћућеТЏ┤тіат╣┐Т│Џ№╝їТЅђС╗Ц№╝їТѕЉУДЅтЙЌСИђУѕгТѕЉС╗гТЈљ Dubbo уџёУ»Ю№╝їтцДжЃетѕєТў»тѕєтИЃт╝Ју│╗у╗ЪуџёТЃЁтєхсђѓ

ТѕЉС╗гтѕџтѕџТЈљтѕ░С║єтѕєтИЃт╝ЈУ┐ЎСИфТдѓт┐х№╝їСИІжЮбтєЇу╗ЎтцДт«ХС╗Іу╗ЇСИђСИІС╗ђС╣ѕТў»тѕєтИЃт╝Ј№╝ЪСИ║С╗ђС╣ѕУдЂтѕєтИЃт╝Ј№╝Ъ

# 1.4 С╗ђС╣ѕТў»тѕєтИЃт╝Ј?

тѕєтИЃт╝ЈТѕќУђЁУ»┤ SOA тѕєтИЃт╝ЈжЄЇУдЂуџёт░▒Тў»жЮбтљЉТюЇтіА№╝їУ»┤у«ђтЇЋуџётѕєтИЃт╝Јт░▒Тў»ТѕЉС╗гТііТЋ┤СИфу│╗у╗ЪТІєтѕєТѕљСИЇтљїуџёТюЇтіАуёХтљјт░єУ┐ЎС║ЏТюЇтіАТћЙтюеСИЇтљїуџёТюЇтіАтЎеСИітЄЈУй╗тЇЋСйЊТюЇтіАуџётјІтіЏТЈљжФўт╣ХтЈЉжЄЈтњїТђДУЃйсђѓТ»ћтдѓућхтЋєу│╗у╗ЪтЈ»С╗Цу«ђтЇЋтю░ТІєтѕєТѕљУ«бтЇЋу│╗у╗ЪсђЂтЋєтЊЂу│╗у╗ЪсђЂуЎ╗тйЋу│╗у╗ЪуГЅуГЅ№╝їТІєтѕєС╣ІтљјуџёТ»ЈСИфТюЇтіАтЈ»С╗ЦжЃеуй▓тюеСИЇтљїуџёТю║тЎеСИі№╝їтдѓТъюТЪљСИђСИфТюЇтіАуџёУ«┐жЌ«жЄЈТ»ћУЙЃтцДуџёУ»ЮС╣ЪтЈ»С╗Цт░єУ┐ЎСИфТюЇтіАтљїТЌХжЃеуй▓тюетцџтЈ░Тю║тЎеСИісђѓ

# 1.5 СИ║С╗ђС╣ѕУдЂтѕєтИЃт╝Ј?

С╗јт╝ђтЈЉУДњт║дТЮЦУ«▓тЇЋСйЊт║ћућеуџёС╗БуаЂжЃйжЏєСИГтюеСИђУхи№╝їУђїтѕєтИЃт╝Ју│╗у╗ЪуџёС╗БуаЂТа╣ТЇ«СИџтіАУбФТІєтѕєсђѓТЅђС╗Ц№╝їТ»ЈСИфтЏбжўЪтЈ»С╗ЦУ┤ЪУ┤БСИђСИфТюЇтіАуџёт╝ђтЈЉ№╝їУ┐ЎТаиТЈљтЇЄС║єт╝ђтЈЉТЋѕујЄсђѓтЈдтцќ№╝їС╗БуаЂТа╣ТЇ«СИџтіАТІєтѕєС╣ІтљјТЏ┤тіаСЙ┐С║ју╗┤ТіцтњїТЅЕт▒Ћсђѓ

тЈдтцќ№╝їТѕЉУДЅтЙЌт░єу│╗у╗ЪТІєтѕєТѕљтѕєтИЃт╝ЈС╣ІтљјСИЇтЁЅСЙ┐С║ју│╗у╗ЪТЅЕт▒Ћтњїу╗┤Тіц№╝їТЏ┤УЃйТЈљжФўТЋ┤СИфу│╗у╗ЪуџёТђДУЃйсђѓСйаТЃ│СИђТЃ│тўЏ?ТііТЋ┤СИфу│╗у╗ЪТІєтѕєТѕљСИЇтљїуџёТюЇтіА/у│╗у╗Ъ№╝їуёХтљјТ»ЈСИфТюЇтіА/у│╗у╗Ъ тЇЋуІгжЃеуй▓тюеСИђтЈ░ТюЇтіАтЎеСИі№╝їТў»СИЇТў»тЙѕтцДуеІт║дСИіТЈљжФўС║єу│╗у╗ЪТђДУЃйтЉб№╝Ъ

# С║ї Dubbo уџёТъХТъё

# 2.1 Dubbo уџёТъХТъётЏЙУДБ

СИіУ┐░Уіѓуѓ╣у«ђтЇЋУ»┤Тўј№╝џ

  • Provider№╝џ Тџ┤жю▓ТюЇтіАуџёТюЇтіАТЈљСЙЏТќ╣
  • Consumer№╝џ У░ЃућеУ┐юуеІТюЇтіАуџёТюЇтіАТХѕУ┤╣Тќ╣
  • Registry№╝џ ТюЇтіАТ│етєїСИјтЈЉуј░уџёТ│етєїСИГт┐Ѓ
  • Monitor№╝џ у╗ЪУ«АТюЇтіАуџёУ░ЃућеТгАТЋ░тњїУ░ЃућеТЌХжЌ┤уџёуЏЉТјДСИГт┐Ѓ
  • Container№╝џ ТюЇтіАУ┐љУАїт«╣тЎе

У░ЃућетЁ│у│╗У»┤Тўј№╝џ

  1. ТюЇтіАт«╣тЎеУ┤ЪУ┤Бтљ»тіе№╝їтіаУйй№╝їУ┐љУАїТюЇтіАТЈљСЙЏУђЁсђѓ
  2. ТюЇтіАТЈљСЙЏУђЁтюетљ»тіеТЌХ№╝їтљЉТ│етєїСИГт┐ЃТ│етєїУЄфти▒ТЈљСЙЏуџёТюЇтіАсђѓ
  3. ТюЇтіАТХѕУ┤╣УђЁтюетљ»тіеТЌХ№╝їтљЉТ│етєїСИГт┐ЃУ«бжўЁУЄфти▒ТЅђжюђуџёТюЇтіАсђѓ
  4. Т│етєїСИГт┐ЃУ┐ћтЏъТюЇтіАТЈљСЙЏУђЁтю░тЮђтѕЌУАеу╗ЎТХѕУ┤╣УђЁ№╝їтдѓТъюТюЅтЈўТЏ┤№╝їТ│етєїСИГт┐Ѓт░єтЪ║С║јжЋ┐У┐ъТјЦТјежђЂтЈўТЏ┤ТЋ░ТЇ«у╗ЎТХѕУ┤╣УђЁсђѓ
  5. ТюЇтіАТХѕУ┤╣УђЁ№╝їС╗јТЈљСЙЏУђЁтю░тЮђтѕЌУАеСИГ№╝їтЪ║С║јУй»У┤ЪУййтЮЄУААу«ЌТ│Ћ№╝їжђЅСИђтЈ░ТЈљСЙЏУђЁУ┐ЏУАїУ░Ѓуће№╝їтдѓТъюУ░Ѓућетц▒У┤Ц№╝їтєЇжђЅтЈдСИђтЈ░У░Ѓућесђѓ
  6. ТюЇтіАТХѕУ┤╣УђЁтњїТЈљСЙЏУђЁ№╝їтюетєЁтГўСИГу┤»У«АУ░ЃућеТгАТЋ░тњїУ░ЃућеТЌХжЌ┤№╝їт«џТЌХТ»ЈтѕєжњЪтЈЉжђЂСИђТгАу╗ЪУ«АТЋ░ТЇ«тѕ░уЏЉТјДСИГт┐Ѓсђѓ

жЄЇУдЂуЪЦУ»єуѓ╣Тђ╗у╗Њ№╝џ

  • Т│етєїСИГт┐ЃУ┤ЪУ┤БТюЇтіАтю░тЮђуџёТ│етєїСИјТЪЦТЅЙ№╝їуЏИтйЊС║јуЏ«тйЋТюЇтіА№╝їТюЇтіАТЈљСЙЏУђЁтњїТХѕУ┤╣УђЁтЈфтюетљ»тіеТЌХСИјТ│етєїСИГт┐ЃС║цС║њ№╝їТ│етєїСИГт┐ЃСИЇУйгтЈЉУ»иТ▒ѓ№╝їтјІтіЏУЙЃт░Ј
  • уЏЉТјДСИГт┐ЃУ┤ЪУ┤Бу╗ЪУ«АтљёТюЇтіАУ░ЃућеТгАТЋ░№╝їУ░ЃућеТЌХжЌ┤уГЅ№╝їу╗ЪУ«АтЁѕтюетєЁтГўТ▒ЄТђ╗тљјТ»ЈтѕєжњЪСИђТгАтЈЉжђЂтѕ░уЏЉТјДСИГт┐ЃТюЇтіАтЎе№╝їт╣ХС╗ЦТіЦУАет▒Ћуц║
  • Т│етєїСИГт┐Ѓ№╝їТюЇтіАТЈљСЙЏУђЁ№╝їТюЇтіАТХѕУ┤╣УђЁСИЅУђЁС╣ІжЌ┤тЮЄСИ║жЋ┐У┐ъТјЦ№╝їуЏЉТјДСИГт┐ЃжЎцтцќ
  • Т│етєїСИГт┐ЃжђџУ┐ЄжЋ┐У┐ъТјЦТёЪуЪЦТюЇтіАТЈљСЙЏУђЁуџётГўтюе№╝їТюЇтіАТЈљСЙЏУђЁт«ЋТю║№╝їТ│етєїСИГт┐Ѓт░єуФІтЇ│ТјежђЂС║ІС╗ХжђџуЪЦТХѕУ┤╣УђЁ
  • Т│етєїСИГт┐ЃтњїуЏЉТјДСИГт┐ЃтЁежЃет«ЋТю║№╝їСИЇтй▒тЊЇти▓У┐љУАїуџёТЈљСЙЏУђЁтњїТХѕУ┤╣УђЁ№╝їТХѕУ┤╣УђЁтюеТюгтю░у╝ЊтГўС║єТЈљСЙЏУђЁтѕЌУАе
  • Т│етєїСИГт┐ЃтњїуЏЉТјДСИГт┐ЃжЃйТў»тЈ»жђЅуџё№╝їТюЇтіАТХѕУ┤╣УђЁтЈ»С╗ЦуЏ┤У┐ъТюЇтіАТЈљСЙЏУђЁ
  • ТюЇтіАТЈљСЙЏУђЁТЌауіХТђЂ№╝їС╗╗ТёЈСИђтЈ░т«ЋТјЅтљј№╝їСИЇтй▒тЊЇСй┐уће
  • ТюЇтіАТЈљСЙЏУђЁтЁежЃет«ЋТјЅтљј№╝їТюЇтіАТХѕУ┤╣УђЁт║ћућет░єТЌаТ│ЋСй┐уће№╝їт╣ХТЌажЎљТгАжЄЇУ┐ъуГЅтЙЁТюЇтіАТЈљСЙЏУђЁТЂбтцЇ

# 2.2 Dubbo тиЦСйютјЪуљє

тЏЙСИГС╗јСИІУЄ│СИітѕєСИ║тЇЂт▒ѓ№╝їтљёт▒ѓтЮЄСИ║тЇЋтљЉСЙЮУхќ№╝їтЈ│УЙ╣уџёж╗ЉУЅ▓у«Гтц┤С╗БУАет▒ѓС╣ІжЌ┤уџёСЙЮУхќтЁ│у│╗№╝їТ»ЈСИђт▒ѓжЃйтЈ»С╗ЦтЅЦуд╗СИіт▒ѓУбФтцЇуће№╝їтЁХСИГ№╝їService тњї Config т▒ѓСИ║ API№╝їтЁХт«Ѓтљёт▒ѓтЮЄСИ║ SPIсђѓ

тљёт▒ѓУ»┤Тўј№╝џ

  • уггСИђт▒ѓ№╝џserviceт▒ѓ№╝їТјЦтЈБт▒ѓ№╝їу╗ЎТюЇтіАТЈљСЙЏУђЁтњїТХѕУ┤╣УђЁТЮЦт«ъуј░уџё
  • уггС║їт▒ѓ№╝џconfigт▒ѓ№╝їжЁЇуй«т▒ѓ№╝їСИ╗УдЂТў»т»╣dubboУ┐ЏУАїтљёуДЇжЁЇуй«уџё
  • уггСИЅт▒ѓ№╝џproxyт▒ѓ№╝їТюЇтіАТјЦтЈБжђЈТўјС╗Буљє№╝їућЪТѕљТюЇтіАуџёт«бТѕиуФ» Stub тњїТюЇтіАтЎеуФ» Skeleton
  • уггтЏЏт▒ѓ№╝џregistryт▒ѓ№╝їТюЇтіАТ│етєїт▒ѓ№╝їУ┤ЪУ┤БТюЇтіАуџёТ│етєїСИјтЈЉуј░
  • уггС║ћт▒ѓ№╝џclusterт▒ѓ№╝їжЏєуЙцт▒ѓ№╝їт░ЂУБЁтцџСИфТюЇтіАТЈљСЙЏУђЁуџёУи»ућ▒С╗ЦтЈіУ┤ЪУййтЮЄУАА№╝їт░єтцџСИфт«ъСЙІу╗ётљѕТѕљСИђСИфТюЇтіА
  • уггтЁГт▒ѓ№╝џmonitorт▒ѓ№╝їуЏЉТјДт▒ѓ№╝їт»╣rpcТјЦтЈБуџёУ░ЃућеТгАТЋ░тњїУ░ЃућеТЌХжЌ┤У┐ЏУАїуЏЉТјД
  • уггСИЃт▒ѓ№╝џprotocolт▒ѓ№╝їУ┐юуеІУ░Ѓућет▒ѓ№╝їт░ЂУБЁrpcУ░Ѓуће
  • уггтЁФт▒ѓ№╝џexchangeт▒ѓ№╝їС┐АТЂ»С║цТЇбт▒ѓ№╝їт░ЂУБЁУ»иТ▒ѓтЊЇт║ћТеАт╝Ј№╝їтљїТГЦУйгт╝ѓТГЦ
  • уггС╣Ют▒ѓ№╝џtransportт▒ѓ№╝їуйЉу╗юС╝аУЙЊт▒ѓ№╝їТійУ▒АminaтњїnettyСИ║у╗ЪСИђТјЦтЈБ
  • уггтЇЂт▒ѓ№╝џserializeт▒ѓ№╝їТЋ░ТЇ«т║ЈтѕЌтїќт▒ѓсђѓуйЉу╗юС╝аУЙЊжюђУдЂсђѓ

# СИЅ Dubbo уџёУ┤ЪУййтЮЄУААуГќуЋЦ

# 3.1 тЁѕТЮЦУДБжЄіСИђСИІС╗ђС╣ѕТў»У┤ЪУййтЮЄУАА

тЁѕТЮЦСИфт«ўТќ╣уџёУДБжЄісђѓ

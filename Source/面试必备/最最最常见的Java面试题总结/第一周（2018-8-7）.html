ждќтЁѕтЏъжАЙСИђСИІтюеуеІт║ЈУ«ЙУ«АУ»ГУеђСИГТюЅтЁ│т░єтЈѓТЋ░С╝ажђњу╗ЎТќ╣Т│Ћ№╝ѕТѕќтЄйТЋ░№╝ЅуџёСИђС║ЏСИЊСИџТю»У»ГсђѓТїЅтђ╝У░Ѓуће(call by value)УАеуц║Тќ╣Т│ЋТјЦТћХуџёТў»У░ЃућеУђЁТЈљСЙЏуџётђ╝№╝їУђїТїЅт╝ЋућеУ░Ѓуће№╝ѕcall by reference)УАеуц║Тќ╣Т│ЋТјЦТћХуџёТў»У░ЃућеУђЁТЈљСЙЏуџётЈўжЄЈтю░тЮђсђѓСИђСИфТќ╣Т│ЋтЈ»С╗ЦС┐«Тћ╣С╝ажђњт╝ЋућеТЅђт»╣т║ћуџётЈўжЄЈтђ╝№╝їУђїСИЇУЃйС┐«Тћ╣С╝ажђњтђ╝У░ЃућеТЅђт»╣т║ћуџётЈўжЄЈтђ╝сђѓ т«ЃућеТЮЦТЈЈУ┐░тљёуДЇуеІт║ЈУ«ЙУ«АУ»ГУеђ№╝ѕСИЇтЈфТў»Java)СИГТќ╣Т│ЋтЈѓТЋ░С╝ажђњТќ╣т╝Јсђѓ

JavaуеІт║ЈУ«ЙУ«АУ»ГУеђТђ╗Тў»жЄЄућеТїЅтђ╝У░ЃућесђѓС╣Ът░▒Тў»У»┤№╝їТќ╣Т│ЋтЙЌтѕ░уџёТў»ТЅђТюЅтЈѓТЋ░тђ╝уџёСИђСИфТІиУ┤Ю№╝їС╣Ът░▒Тў»У»┤№╝їТќ╣Т│ЋСИЇУЃйС┐«Тћ╣С╝ажђњу╗Ўт«ЃуџёС╗╗СйЋтЈѓТЋ░тЈўжЄЈуџётєЁт«╣сђѓ

# example 1

public static void main(String[] args) {
    int num1 = 10;
    int num2 = 20;

    swap(num1, num2);

    System.out.println("num1 = " + num1);
    System.out.println("num2 = " + num2);
}

public static void swap(int a, int b) {
    int temp = a;
    a = b;
    b = temp;

    System.out.println("a = " + a);
    System.out.println("b = " + b);
}

hashCode() уџёСйюућеТў»УјитЈќтЊѕтИїуаЂ№╝їС╣ЪуД░СИ║ТЋБтѕЌуаЂ№╝Џт«Ѓт«ъжЎЁСИіТў»У┐ћтЏъСИђСИфintТЋ┤ТЋ░сђѓУ┐ЎСИфтЊѕтИїуаЂуџёСйюућеТў»уА«т«џУ»Цт»╣У▒АтюетЊѕтИїУАеСИГуџёу┤бт╝ЋСйЇуй«сђѓhashCode() т«џС╣ЅтюеJDKуџёObject.javaСИГ№╝їУ┐Ўт░▒ТёЈтЉ│уЮђJavaСИГуџёС╗╗СйЋу▒╗жЃйтїЁтљФТюЅhashCode() тЄйТЋ░сђѓтЈдтцќжюђУдЂТ│еТёЈуџёТў»№╝џ Object уџё hashcode Тќ╣Т│ЋТў»Тюгтю░Тќ╣Т│Ћ№╝їС╣Ът░▒Тў»уће c У»ГУеђТѕќ c++ т«ъуј░уџё№╝їУ»ЦТќ╣Т│ЋжђџтИИућеТЮЦт░єт»╣У▒Ауџё тєЁтГўтю░тЮђ УйгТЇбСИ║ТЋ┤ТЋ░С╣ІтљјУ┐ћтЏъсђѓ

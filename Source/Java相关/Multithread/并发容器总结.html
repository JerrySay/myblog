  • **ConcurrentLinkedQueue№╝џ**жФўТЋѕуџёт╣ХтЈЉжўЪтѕЌ№╝їСй┐ућежЊЙУАет«ъуј░сђѓтЈ»С╗ЦуюІтЂџСИђСИфу║┐уеІт«ЅтЁеуџё LinkedList№╝їУ┐ЎТў»СИђСИфжЮъжў╗тАъжўЪтѕЌсђѓ
  • BlockingQueue: У┐ЎТў»СИђСИфТјЦтЈБ№╝їJDKтєЁжЃежђџУ┐ЄжЊЙУАесђЂТЋ░у╗ёуГЅТќ╣т╝Јт«ъуј░С║єУ┐ЎСИфТјЦтЈБсђѓУАеуц║жў╗тАъжўЪтѕЌ№╝їжЮътИИжђѓтљѕућеС║јСйюСИ║ТЋ░ТЇ«тЁ▒С║ФуџёжђџжЂЊсђѓ

# С║ї ConcurrentHashMap

ТѕЉС╗гуЪЦжЂЊ HashMap СИЇТў»у║┐уеІт«ЅтЁеуџё№╝їтюет╣ХтЈЉтю║ТЎ»СИІтдѓТъюУдЂС┐ЮУ»ЂСИђуДЇтЈ»УАїуџёТќ╣т╝ЈТў»Сй┐уће Collections.synchronizedMap() Тќ╣Т│ЋТЮЦтїЁУБЁТѕЉС╗гуџё HashMapсђѓСйєУ┐ЎТў»жђџУ┐ЄСй┐ућеСИђСИфтЁет▒ђуџёжћЂТЮЦтљїТГЦСИЇтљїу║┐уеІжЌ┤уџёт╣ХтЈЉУ«┐жЌ«№╝їтЏаТГцС╝џтИдТЮЦСИЇтЈ»т┐йУДєуџёТђДУЃйжЌ«жбўсђѓ

ТЅђС╗Цт░▒ТюЅС║є HashMap уџёу║┐уеІт«ЅтЁеуЅѕТюгРђћРђћ ConcurrentHashMap уџёУ»ъућЪсђѓтюеConcurrentHashMapСИГ№╝їТЌаУ«║Тў»У»╗ТЊЇСйюУ┐ўТў»тєЎТЊЇСйюжЃйУЃйС┐ЮУ»ЂтЙѕжФўуџёТђДУЃй№╝џтюеУ┐ЏУАїУ»╗ТЊЇСйюТЌХ(тЄаС╣ј)СИЇжюђУдЂтіажћЂ№╝їУђїтюетєЎТЊЇСйюТЌХжђџУ┐ЄжћЂтѕєТ«хТіђТю»тЈфт»╣ТЅђТЊЇСйюуџёТ«хтіажћЂУђїСИЇтй▒тЊЇт«бТѕиуФ»т»╣тЁХт«ЃТ«хуџёУ«┐жЌ«сђѓ

тЁ│С║ј ConcurrentHashMap уЏИтЁ│жЌ«жбў№╝їТѕЉтюе сђіУ┐ЎтЄажЂЊJavaжЏєтљѕТАєТъХжЮбУ»ЋжбўтЄаС╣јт┐ЁжЌ«сђІ У┐Ўу»ЄТќЄуФаСИГти▓у╗ЈТЈљтѕ░У┐ЄсђѓСИІжЮбТб│уљєСИђСИІтЁ│С║ј ConcurrentHashMap Т»ћУЙЃжЄЇУдЂуџёжЌ«жбў№╝џ

  • ConcurrentHashMap тњї Hashtable уџётї║тѕФ
  • ConcurrentHashMapу║┐уеІт«ЅтЁеуџётЁиСйЊт«ъуј░Тќ╣т╝Ј/т║Ћт▒ѓтЁиСйЊт«ъуј░

# 3.2 CopyOnWriteArrayList Тў»тдѓСйЋтЂџтѕ░уџё№╝Ъ

CopyOnWriteArrayList у▒╗уџёТЅђТюЅтЈ»тЈўТЊЇСйю№╝ѕadd№╝їsetуГЅуГЅ№╝ЅжЃйТў»жђџУ┐ЄтѕЏт╗║т║Ћт▒ѓТЋ░у╗ёуџёТќ░тЅ»ТюгТЮЦт«ъуј░уџёсђѓтйЊ List жюђУдЂУбФС┐«Тћ╣уџёТЌХтђЎ№╝їТѕЉт╣ХСИЇС┐«Тћ╣тјЪТюЅтєЁт«╣№╝їУђїТў»т»╣тјЪТюЅТЋ░ТЇ«У┐ЏУАїСИђТгАтцЇтѕХ№╝їт░єС┐«Тћ╣уџётєЁт«╣тєЎтЁЦтЅ»ТюгсђѓтєЎт«їС╣Ітљј№╝їтєЇт░єС┐«Тћ╣т«їуџётЅ»ТюгТЏ┐ТЇбтјЪТЮЦуџёТЋ░ТЇ«№╝їУ┐ЎТаит░▒тЈ»С╗ЦС┐ЮУ»ЂтєЎТЊЇСйюСИЇС╝џтй▒тЊЇУ»╗ТЊЇСйюС║єсђѓ

С╗ј CopyOnWriteArrayList уџётљЇтГЌт░▒УЃйуюІтЄ║CopyOnWriteArrayList Тў»Т╗АУХ│CopyOnWrite уџёArrayList№╝їТЅђУ░ЊCopyOnWrite С╣Ът░▒Тў»У»┤№╝џтюеУ«Ау«ЌТю║№╝їтдѓТъюСйаТЃ│УдЂт»╣СИђтЮЌтєЁтГўУ┐ЏУАїС┐«Тћ╣ТЌХ№╝їТѕЉС╗гСИЇтюетјЪТюЅтєЁтГўтЮЌСИГУ┐ЏУАїтєЎТЊЇСйю№╝їУђїТў»т░єтєЁтГўТІиУ┤ЮСИђС╗й№╝їтюеТќ░уџётєЁтГўСИГУ┐ЏУАїтєЎТЊЇСйю№╝їтєЎт«їС╣ІтљјтЉб№╝їт░▒т░єТїЄтљЉтјЪТЮЦтєЁтГўТїЄжњѕТїЄтљЉТќ░уџётєЁтГў№╝їтјЪТЮЦуџётєЁтГўт░▒тЈ»С╗ЦУбФтЏъТћХТјЅС║єсђѓ

# 3.3.2 CopyOnWriteArrayList тєЎтЁЦТЊЇСйюуџёт«ъуј░

CopyOnWriteArrayList тєЎтЁЦТЊЇСйю add() Тќ╣Т│ЋтюеТи╗тіажЏєтљѕуџёТЌХтђЎтіаС║єжћЂ№╝їС┐ЮУ»ЂС║єтљїТГЦ№╝їжЂ┐тЁЇС║єтцџу║┐уеІтєЎуџёТЌХтђЎС╝џ copy тЄ║тцџСИфтЅ»ТюгтЄ║ТЮЦсђѓ

    /**
     * Appends the specified element to the end of this list.
     *
     * @param e element to be appended to this list
     * @return {@code true} (as specified by {@link Collection#add})
     */
    public boolean add(E e) {
        final ReentrantLock lock = this.lock;
        lock.lock();//тіажћЂ
        try {
            Object[] elements = getArray();
            int len = elements.length;
            Object[] newElements = Arrays.copyOf(elements, len + 1);//ТІиУ┤ЮТќ░ТЋ░у╗ё
            newElements[len] = e;
            setArray(newElements);
            return true;
        } finally {
            lock.unlock();//жЄіТћЙжћЂ
        }
    }

# тЏЏ ConcurrentLinkedQueue

JavaТЈљСЙЏуџёу║┐уеІт«ЅтЁеуџё Queue тЈ»С╗ЦтѕєСИ║жў╗тАъжўЪтѕЌтњїжЮъжў╗тАъжўЪтѕЌ№╝їтЁХСИГжў╗тАъжўЪтѕЌуџётЁИтъІСЙІтГљТў» BlockingQueue№╝їжЮъжў╗тАъжўЪтѕЌуџётЁИтъІСЙІтГљТў»ConcurrentLinkedQueue№╝їтюет«ъжЎЁт║ћућеСИГУдЂТа╣ТЇ«т«ъжЎЁжюђУдЂжђЅућежў╗тАъжўЪтѕЌТѕќУђЁжЮъжў╗тАъжўЪтѕЌсђѓ жў╗тАъжўЪтѕЌтЈ»С╗ЦжђџУ┐ЄтіажћЂТЮЦт«ъуј░№╝їжЮъжў╗тАъжўЪтѕЌтЈ»С╗ЦжђџУ┐Є CAS ТЊЇСйют«ъуј░сђѓ

С╗јтљЇтГЌтЈ»С╗ЦуюІтЄ║№╝їConcurrentLinkedQueueУ┐ЎСИфжўЪтѕЌСй┐ућежЊЙУАеСйюСИ║тЁХТЋ░ТЇ«у╗ЊТъё№╝јConcurrentLinkedQueue т║ћУ»Цу«ЌТў»тюежФўт╣ХтЈЉуј»тбЃСИГТђДУЃйТюђтЦйуџёжўЪтѕЌС║єсђѓт«ЃС╣ІТЅђТюЅУЃйТюЅтЙѕтЦйуџёТђДУЃй№╝їТў»тЏаСИ║тЁХтєЁжЃетцЇТЮѓуџёт«ъуј░сђѓ

ConcurrentLinkedQueue тєЁжЃеС╗БуаЂТѕЉС╗гт░▒СИЇтѕєТъљС║є№╝їтцДт«ХуЪЦжЂЊConcurrentLinkedQueue СИ╗УдЂСй┐уће CAS жЮъжў╗тАъу«ЌТ│ЋТЮЦт«ъуј░у║┐уеІт«ЅтЁет░▒тЦйС║єсђѓ

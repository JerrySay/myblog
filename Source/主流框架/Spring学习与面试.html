# AOP№╝џ

AOPТђЮТЃ│уџёт«ъуј░СИђУѕгжЃйТў»тЪ║С║ј С╗БуљєТеАт╝Ј №╝їтюеJAVAСИГСИђУѕгжЄЄућеJDKтіеТђЂС╗БуљєТеАт╝Ј№╝їСйєТў»ТѕЉС╗гжЃйуЪЦжЂЊ№╝їJDKтіеТђЂС╗БуљєТеАт╝ЈтЈфУЃйС╗БуљєТјЦтЈБУђїСИЇУЃйС╗Буљєу▒╗сђѓтЏаТГц№╝їSpring AOP С╝џУ┐ЎТаитГљТЮЦУ┐ЏУАїтѕЄТЇб№╝їтЏаСИ║Spring AOP тљїТЌХТћ»ТїЂ CGLIBсђЂASPECTJсђЂJDKтіеТђЂС╗Буљєсђѓ

  • тдѓТъюуЏ«ТаЄт»╣У▒Ауџёт«ъуј░у▒╗т«ъуј░С║єТјЦтЈБ№╝їSpring AOP т░єС╝џжЄЄуће JDK тіеТђЂС╗БуљєТЮЦућЪТѕљ AOP С╗Буљєу▒╗№╝Џ
  • тдѓТъюуЏ«ТаЄт»╣У▒Ауџёт«ъуј░у▒╗Т▓АТюЅт«ъуј░ТјЦтЈБ№╝їSpring AOP т░єС╝џжЄЄуће CGLIB ТЮЦућЪТѕљ AOP С╗Буљєу▒╗РђћРђћСИЇУ┐ЄУ┐ЎСИфжђЅТІЕУ┐ЄуеІт»╣т╝ђтЈЉУђЁт«їтЁежђЈТўјсђЂт╝ђтЈЉУђЁС╣ЪТЌажюђтЁ│т┐Ѓсђѓ

Рђ╗жЮЎТђЂС╗БуљєсђЂJDKтіеТђЂС╗БуљєсђЂCGLIBтіеТђЂС╗БуљєУ«▓УДБ

ТѕЉС╗гуЪЦжЂЊAOPТђЮТЃ│уџёт«ъуј░СИђУѕгжЃйТў»тЪ║С║ј С╗БуљєТеАт╝Ј №╝їТЅђС╗ЦтюеуюІСИІжЮбуџёТќЄуФаС╣ІтЅЇт╗║У««тЁѕС║єУДБСИђСИІжЮЎТђЂС╗БуљєС╗ЦтЈіJDKтіеТђЂС╗БуљєсђЂCGLIBтіеТђЂС╗Буљєуџёт«ъуј░Тќ╣т╝Јсђѓ

Spring AOP тЁЦжЌе

тИдСйатЁЦжЌеуџёСИђу»ЄТќЄуФасђѓУ┐Ўу»ЄТќЄуФаСИ╗УдЂС╗Іу╗ЇС║єAOPСИГуџётЪ║ТюгТдѓт┐х№╝џ5уДЇу▒╗тъІуџёжђџуЪЦ№╝ѕBefore№╝їAfter№╝їAfter-returning№╝їAfter-throwing№╝їAround№╝Ѕ№╝ЏSpringСИГт»╣AOPуџёТћ»ТїЂ№╝џAOPТђЮТЃ│уџёт«ъуј░СИђУѕгжЃйТў»тЪ║С║јС╗БуљєТеАт╝Ј№╝їтюеJAVAСИГСИђУѕгжЄЄућеJDKтіеТђЂС╗БуљєТеАт╝Ј№╝їSpring AOP тљїТЌХТћ»ТїЂ CGLIBсђЂASPECTJсђЂJDKтіеТђЂС╗Буљє№╝ї

Рђ╗Spring AOP тЪ║С║јAspectJТ│еУДБтдѓСйЋт«ъуј░AOP

AspectJТў»СИђСИфAOPТАєТъХ№╝їт«ЃУЃйтцЪт»╣javaС╗БуаЂУ┐ЏУАїAOPу╝ќУ»Љ№╝ѕСИђУѕгтюеу╝ќУ»ЉТюЪУ┐ЏУАї№╝Ѕ№╝їУ«ЕjavaС╗БуаЂтЁиТюЅAspectJуџёAOPтіЪУЃй№╝ѕтйЊуёХжюђУдЂуЅ╣Т«іуџёу╝ќУ»ЉтЎе№╝Ѕ№╝їтЈ»С╗ЦУ┐ЎТаиУ»┤AspectJТў»уЏ«тЅЇт«ъуј░AOPТАєТъХСИГТюђТѕљуєЪ№╝їтіЪУЃйТюђСИ░т»їуџёУ»ГУеђ№╝їТЏ┤т╣ИУ┐љуџёТў»№╝їAspectJСИјjavaуеІт║Јт«їтЁетЁ╝т«╣№╝їтЄаС╣јТў»ТЌау╝ЮтЁ│УЂћ№╝їтЏаТГцт»╣С║јТюЅjavaу╝ќуеІтЪ║уАђуџётиЦуеІтИѕ№╝їСИіТЅІтњїСй┐ућежЃйжЮътИИт«╣ТўЊ

SpringТ│еТёЈтѕ░AspectJтюеAOPуџёт«ъуј░Тќ╣т╝ЈСИіСЙЮУхќС║јуЅ╣Т«іу╝ќУ»ЉтЎе(ajcу╝ќУ»ЉтЎе)№╝їтЏаТГцSpringтЙѕТю║ТЎ║тЏъжЂ┐С║єУ┐Ўуѓ╣№╝їУйгтљЉжЄЄућетіеТђЂС╗БуљєТіђТю»уџёт«ъуј░тјЪуљєТЮЦТъёт╗║Spring AOPуџётєЁжЃеТю║тѕХ№╝ѕтіеТђЂу╗ЄтЁЦ№╝Ѕ№╝їУ┐ЎТў»СИјAspectJ№╝ѕжЮЎТђЂу╗ЄтЁЦ№╝ЅТюђТа╣Тюгуџётї║тѕФсђѓ

Рђ╗ТјбуДўSpring AOP№╝ѕТЁЋУ»ЙуйЉУДєжбЉ№╝їтЙѕСИЇжћЎ№╝Ѕ

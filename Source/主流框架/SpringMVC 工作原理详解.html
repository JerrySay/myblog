# SpringMVC у«ђтЇЋС╗Іу╗Ї

SpringMVC ТАєТъХТў»С╗ЦУ»иТ▒ѓСИ║жЕ▒тіе№╝їтЏ┤у╗Ћ Servlet У«ЙУ«А№╝їт░єУ»иТ▒ѓтЈЉу╗ЎТјДтѕХтЎе№╝їуёХтљјжђџУ┐ЄТеАтъІт»╣У▒А№╝їтѕєТ┤ЙтЎеТЮЦт▒Ћуц║У»иТ▒ѓу╗ЊТъюУДєтЏЙсђѓтЁХСИГТаИт┐Ѓу▒╗Тў» DispatcherServlet№╝їт«ЃТў»СИђСИф Servlet№╝їжАХт▒ѓТў»т«ъуј░уџёServletТјЦтЈБсђѓ

# SpringMVC тиЦСйютјЪуљє№╝ѕжЄЇУдЂ№╝Ѕ

у«ђтЇЋТЮЦУ»┤№╝џ

т«бТѕиуФ»тЈЉжђЂУ»иТ▒ѓ-> тЅЇуФ»ТјДтѕХтЎе DispatcherServlet ТјЦтЈЌт«бТѕиуФ»У»иТ▒ѓ -> ТЅЙтѕ░тцёуљєтЎеТўат░ё HandlerMapping УДБТъљУ»иТ▒ѓт»╣т║ћуџё Handler-> HandlerAdapter С╝џТа╣ТЇ« Handler ТЮЦУ░ЃућеуюЪТГБуџётцёуљєтЎет╝ђтцёуљєУ»иТ▒ѓ№╝їт╣ХтцёуљєуЏИт║ћуџёСИџтіАжђ╗УЙЉ -> тцёуљєтЎеУ┐ћтЏъСИђСИфТеАтъІУДєтЏЙ ModelAndView -> УДєтЏЙУДБТъљтЎеУ┐ЏУАїУДБТъљ -> У┐ћтЏъСИђСИфУДєтЏЙт»╣У▒А->тЅЇуФ»ТјДтѕХтЎе DispatcherServlet ТИ▓ТЪЊТЋ░ТЇ«№╝ѕModer№╝Ѕ->т░єтЙЌтѕ░УДєтЏЙт»╣У▒АУ┐ћтЏъу╗ЎућеТѕи

тдѓСИІтЏЙТЅђуц║№╝џ

СИітЏЙуџёСИђСИфугћУ»»уџёт░ЈжЌ«жбў№╝џSpring MVC уџётЁЦтЈБтЄйТЋ░С╣Ът░▒Тў»тЅЇуФ»ТјДтѕХтЎе DispatcherServlet уџёСйюућеТў»ТјЦТћХУ»иТ▒ѓ№╝їтЊЇт║ћу╗ЊТъюсђѓ

ТхЂуеІУ»┤Тўј№╝ѕжЄЇУдЂ№╝Ѕ№╝џ

№╝ѕ1№╝Ѕт«бТѕиуФ»№╝ѕТхЈУДѕтЎе№╝ЅтЈЉжђЂУ»иТ▒ѓ№╝їуЏ┤ТјЦУ»иТ▒ѓтѕ░ DispatcherServletсђѓ

№╝ѕ2№╝ЅDispatcherServlet Та╣ТЇ«У»иТ▒ѓС┐АТЂ»У░Ѓуће HandlerMapping№╝їУДБТъљУ»иТ▒ѓт»╣т║ћуџё Handlerсђѓ

№╝ѕ3№╝ЅУДБТъљтѕ░т»╣т║ћуџё Handler№╝ѕС╣Ът░▒Тў»ТѕЉС╗гт╣│тИИУ»┤уџё Controller ТјДтѕХтЎе№╝Ѕтљј№╝їт╝ђтДІућ▒ HandlerAdapter жђѓжЁЇтЎетцёуљєсђѓ

№╝ѕ4№╝ЅHandlerAdapter С╝џТа╣ТЇ« Handler ТЮЦУ░ЃућеуюЪТГБуџётцёуљєтЎет╝ђтцёуљєУ»иТ▒ѓ№╝їт╣ХтцёуљєуЏИт║ћуџёСИџтіАжђ╗УЙЉсђѓ

№╝ѕ5№╝ЅтцёуљєтЎетцёуљєт«їСИџтіАтљј№╝їС╝џУ┐ћтЏъСИђСИф ModelAndView т»╣У▒А№╝їModel Тў»У┐ћтЏъуџёТЋ░ТЇ«т»╣У▒А№╝їView Тў»СИфжђ╗УЙЉСИіуџё Viewсђѓ

№╝ѕ6№╝ЅViewResolver С╝џТа╣ТЇ«жђ╗УЙЉ View ТЪЦТЅЙт«ъжЎЁуџё Viewсђѓ

№╝ѕ7№╝ЅDispaterServlet ТііУ┐ћтЏъуџё Model С╝ау╗Ў View№╝ѕУДєтЏЙТИ▓ТЪЊ№╝Ѕсђѓ

№╝ѕ8№╝ЅТіі View У┐ћтЏъу╗ЎУ»иТ▒ѓУђЁ№╝ѕТхЈУДѕтЎе№╝Ѕ

# SpringMVC жЄЇУдЂу╗ёС╗ХУ»┤Тўј

1сђЂтЅЇуФ»ТјДтѕХтЎеDispatcherServlet№╝ѕСИЇжюђУдЂтиЦуеІтИѕт╝ђтЈЉ№╝Ѕ,ућ▒ТАєТъХТЈљСЙЏ№╝ѕжЄЇУдЂ№╝Ѕ

Сйюуће№╝џSpring MVC уџётЁЦтЈБтЄйТЋ░сђѓТјЦТћХУ»иТ▒ѓ№╝їтЊЇт║ћу╗ЊТъю№╝їуЏИтйЊС║јУйгтЈЉтЎе№╝їСИГтц«тцёуљєтЎесђѓТюЅС║є DispatcherServlet тЄЈт░ЉС║єтЁХт«Ѓу╗ёС╗ХС╣ІжЌ┤уџёУђдтљѕт║дсђѓућеТѕиУ»иТ▒ѓтѕ░УЙЙтЅЇуФ»ТјДтѕХтЎе№╝їт«Ѓт░▒уЏИтйЊС║јmvcТеАт╝ЈСИГуџёc№╝їDispatcherServletТў»ТЋ┤СИфТхЂуеІТјДтѕХуџёСИГт┐Ѓ№╝їућ▒т«ЃУ░ЃућетЁХт«Ѓу╗ёС╗ХтцёуљєућеТѕиуџёУ»иТ▒ѓ№╝їDispatcherServletуџётГўтюежЎЇСйјС║єу╗ёС╗ХС╣ІжЌ┤уџёУђдтљѕТђДсђѓ

2сђЂтцёуљєтЎеТўат░ётЎеHandlerMapping(СИЇжюђУдЂтиЦуеІтИѕт╝ђтЈЉ),ућ▒ТАєТъХТЈљСЙЏ

Сйюуће№╝џТа╣ТЇ«У»иТ▒ѓуџёurlТЪЦТЅЙHandlerсђѓHandlerMappingУ┤ЪУ┤БТа╣ТЇ«ућеТѕиУ»иТ▒ѓТЅЙтѕ░HandlerтЇ│тцёуљєтЎе№╝ѕController№╝Ѕ№╝їSpringMVCТЈљСЙЏС║єСИЇтљїуџёТўат░ётЎет«ъуј░СИЇтљїуџёТўат░ёТќ╣т╝Ј№╝їСЙІтдѓ№╝џжЁЇуй«ТќЄС╗ХТќ╣т╝Ј№╝їт«ъуј░ТјЦтЈБТќ╣т╝Ј№╝їТ│еУДБТќ╣т╝ЈуГЅсђѓ

3сђЂтцёуљєтЎежђѓжЁЇтЎеHandlerAdapter

Сйюуће№╝џТїЅуЁДуЅ╣т«џУДётѕЎ№╝ѕHandlerAdapterУдЂТ▒ѓуџёУДётѕЎ№╝Ѕтј╗ТЅДУАїHandler жђџУ┐ЄHandlerAdapterт»╣тцёуљєтЎеУ┐ЏУАїТЅДУАї№╝їУ┐ЎТў»жђѓжЁЇтЎеТеАт╝Јуџёт║ћуће№╝їжђџУ┐ЄТЅЕт▒ЋжђѓжЁЇтЎетЈ»С╗Цт»╣ТЏ┤тцџу▒╗тъІуџётцёуљєтЎеУ┐ЏУАїТЅДУАїсђѓ

4сђЂтцёуљєтЎеHandler(жюђУдЂтиЦуеІтИѕт╝ђтЈЉ)

Т│еТёЈ№╝џу╝ќтєЎHandlerТЌХТїЅуЁДHandlerAdapterуџёУдЂТ▒ѓтј╗тЂџ№╝їУ┐ЎТаижђѓжЁЇтЎеТЅЇтЈ»С╗Цтј╗ТГБуА«ТЅДУАїHandler Handler Тў»у╗ДDispatcherServletтЅЇуФ»ТјДтѕХтЎеуџётљјуФ»ТјДтѕХтЎе№╝їтюеDispatcherServletуџёТјДтѕХСИІHandlerт»╣тЁиСйЊуџёућеТѕиУ»иТ▒ѓУ┐ЏУАїтцёуљєсђѓ ућ▒С║јHandlerТХЅтЈітѕ░тЁиСйЊуџёућеТѕиСИџтіАУ»иТ▒ѓ№╝їТЅђС╗ЦСИђУѕгТЃЁтєхжюђУдЂтиЦуеІтИѕТа╣ТЇ«СИџтіАжюђТ▒ѓт╝ђтЈЉHandlerсђѓ

5сђЂУДєтЏЙУДБТъљтЎеView resolver(СИЇжюђУдЂтиЦуеІтИѕт╝ђтЈЉ),ућ▒ТАєТъХТЈљСЙЏ

Сйюуће№╝џУ┐ЏУАїУДєтЏЙУДБТъљ№╝їТа╣ТЇ«жђ╗УЙЉУДєтЏЙтљЇУДБТъљТѕљуюЪТГБуџёУДєтЏЙ№╝ѕview№╝Ѕ View ResolverУ┤ЪУ┤Бт░єтцёуљєу╗ЊТъюућЪТѕљViewУДєтЏЙ№╝їView ResolverждќтЁѕТа╣ТЇ«жђ╗УЙЉУДєтЏЙтљЇУДБТъљТѕљуЅЕуљєУДєтЏЙтљЇтЇ│тЁиСйЊуџёжАхжЮбтю░тЮђ№╝їтєЇућЪТѕљViewУДєтЏЙт»╣У▒А№╝їТюђтљјт»╣ViewУ┐ЏУАїТИ▓ТЪЊт░єтцёуљєу╗ЊТъюжђџУ┐ЄжАхжЮбт▒Ћуц║у╗ЎућеТѕисђѓ springmvcТАєТъХТЈљСЙЏС║єтЙѕтцџуџёViewУДєтЏЙу▒╗тъІ№╝їтїЁТІг№╝џjstlViewсђЂfreemarkerViewсђЂpdfViewуГЅсђѓ СИђУѕгТЃЁтєхСИІжюђУдЂжђџУ┐ЄжАхжЮбТаЄуГЙТѕќжАхжЮбТеАуЅѕТіђТю»т░єТеАтъІТЋ░ТЇ«жђџУ┐ЄжАхжЮбт▒Ћуц║у╗ЎућеТѕи№╝їжюђУдЂућ▒тиЦуеІтИѕТа╣ТЇ«СИџтіАжюђТ▒ѓт╝ђтЈЉтЁиСйЊуџёжАхжЮбсђѓ

6сђЂУДєтЏЙView(жюђУдЂтиЦуеІтИѕт╝ђтЈЉ)

ViewТў»СИђСИфТјЦтЈБ№╝їт«ъуј░у▒╗Тћ»ТїЂСИЇтљїуџёViewу▒╗тъІ№╝ѕjspсђЂfreemarkerсђЂpdf...№╝Ѕ

Т│еТёЈ№╝џтцёуљєтЎеHandler№╝ѕС╣Ът░▒Тў»ТѕЉС╗гт╣│тИИУ»┤уџёControllerТјДтѕХтЎе№╝ЅС╗ЦтЈіУДєтЏЙт▒ѓviewжЃйТў»жюђУдЂТѕЉС╗гУЄфти▒ТЅІтіет╝ђтЈЉуџёсђѓтЁХС╗ќуџёСИђС║Џу╗ёС╗ХТ»ћтдѓ№╝џтЅЇуФ»ТјДтѕХтЎеDispatcherServletсђЂтцёуљєтЎеТўат░ётЎеHandlerMappingсђЂтцёуљєтЎежђѓжЁЇтЎеHandlerAdapterуГЅуГЅжЃйТў»ТАєТъХТЈљСЙЏу╗ЎТѕЉС╗гуџё№╝їСИЇжюђУдЂУЄфти▒ТЅІтіет╝ђтЈЉсђѓ

# DispatcherServletУ»ду╗єУДБТъљ
